# Viewers リリースノート （5.15.3）{#viewers-release-notes}

<!-- Updated January 13, 2021 for the 5.15.3 release-->

## Adobe Dynamic Media Viewers {#section-5460f4e254d44c5a95196e6cb657b7e4}

パッケージには次のものが含まれています。

* **HTML5 ビューアー**

   * BasicZoomViewer
   * eCatalogViewer
   * eCatalogSearchViewer
   * FlyoutViewer
   * MixedMediaViewer
   * SpinViewer
   * VideoViewer
   * ZoomViewer

* **Adobe Experience Manager Dynamic Media Viewers**

   * CarouselViewer
   * DimensionalViewer
   * InteractiveImage
   * InteractiveVideoViewer
   * PanoramicViewer
   * SmartCropVideoViewer
   * Video360Viewer
   * ZoomVerticalViewer

* **HTML5 Viewers for mobile only** （従来、3.10 s7sdkを使用）

   * basicZoomMobile
   * フライアウ
   * genericSpinMobile
   * genericZoomMobile
   * mixedMedia

## 5.15.3の新しい機能強化 {#new-features-and-enhancements-5-15-3}

* SDK バージョンを3.10に更新することで、従来のモバイル HTML5 ビューアのXSSの問題を修正しました。 Dynamic Mediaで使用される新しい標準のHTML5 ビューアは、このXSS問題の影響を受けなかったため、更新されませんでした。

## 5.15.2の新しい機能強化 {#new-features-and-enhancements-5-15-2}

* IOSでのネイティブビデオ再生UIのサポートを修正しました。

## 5.15.1の新機能と機能強化 {#new-features-and-enhancements-5-15-1}

* パッシブイベントリスナーのサポートを追加しました。
* DimensionalViewerの3D レンダリングライブラリを更新し、モデルの色付けをサポートしました。
* 非アクティブなボタンのフォーカスを無効にします。
* 共有ツールでのキーボード制御と支援テクノロジーサポートの改善。
* IOS デバイスでのスウォッチスクロールの動作を修正しました。
* eCatalog ビューアのiOS デバイスでのページ回転効果を修正しました。
* タッチデバイス上のスウォッチ対応ビューアでの`swatchoverlay`修飾子の動作を修正しました。
* メディアセットでビデオが最初に表示されない場合のMixedMediaViewerでのビデオキャプションのサポートを修正しました。

## 既知の問題と制限 {#known-issues-and-restrictions}

* IS コマンドの画像サービング修飾子は、デザインによって`req=set` リクエストに追加されません。 ただし、画像表示のみに影響する修飾子は問題ありません。 サイズに影響を与える修飾子は、複雑なアセットで使用する必要があります。
* ブラウザーウィンドウを拡大・縮小すると、サイズ変更が正しくなくなります。
* iPad 2 - iOSで大規模なe カタログアセットがSafariでクラッシュする。

### すべてのビューア {#all-viewers}

* 透かし、不明化、ロックはサポートされていません。
* 画像プリセットはサポートされていません。
* `display:none` CSSを使用するか、親ノードから動的に分離して、DOMからビューアを追加または削除することはサポートされていません。

### HTML5すべてのビューアー {#html5-all-viewers}

* ビューアをテーブルに埋め込むと、ネイティブ以外のフルスクリーンモードで、ビューアのサイズや配置が正しくない場合があります。 Adobeでは、代わりにDIVを使用することをお勧めします。
* コード内の明示的なインスタンス名を持つパラメーターでは、URL内のインスタンス名も上書きする必要があります（例：`zoomView.iconfeffect=0`）。
* 画像サービングコマンド `crop`はサポートされていません。
* 「閉じる」ボタンは、ビューアが子ウィンドウで開いている場合にのみ機能します。
* IS コマンド修飾子では、画像サイズに影響を与える画像サービング修飾子はサポートされていません。

### HTML5 eCatalog {#html5-ecatalog}

* 他のHTML ページに移動してから戻ると、ビューアが最初のページにリセットされます。
* IOS デバイスをローテーションすると、ページレイアウトが正しく表示されないことがあります。 ページにズームインすると、レイアウトが修正されます。
* 複数ページのスプレッドの左側のページにのみ内部リンクします。 ポートレートモードのモバイルデバイスに影響します。

### HTML5 Mixed Media {#html5-mixed-media}

* サウンドトラックの再生はサポートされていません。

### HTML5 Social {#html5-social}

* 送信メールでサムネールを適切にレンダリングするには、`serverurl`修飾子に絶対URLが必要です。

### HTML5 Video {#html5-video}

* ポスター画像で`max size` エラーが発生する可能性があります。 この場合、画像サービング公開の制限を増やす必要があります。
* ビデオのキャプションには、ホスティング HTML ページが外部サーバー（Dynamic Media サーバーではなく）から提供される場合は、会社のルールセットが必要です。 テクニカルサポートに連絡してサポートを受けてください。
* バッファリングが原因で、Analytics トラッキングで誤った再生率が報告される場合があります。
* IPadまたはAndroid™ デバイスでは、ポスター画像の代わりに黒いフレームが表示されることがあります。
* IPadまたはAndroid™ デバイスでは、ビューアの読み込み中に黒いフレームが画面に点滅することがあります。
* IPad デバイスでは、背景が白/透明に設定されている場合、VideoPlayer コンポーネントの側に黒い境界線が表示されます。
* IPadでは、iOS 7を使用すると、ビデオの最後のフレームが歪む可能性があります。
* Chrome、Firefox、およびInternet Explorerのブラウザーでは、HLS ストリーミングモードでのビデオ検索中に、マクロブロックが発生することがあります。
* 初回訪問者のMicrosoft® Edge ブラウザーにポスター画像が表示されない場合があります。
* プログレッシブ再生を使用すると、Internet Explorer 9でビデオが読み込まれた後、ポスター画像が非表示になる場合があります。

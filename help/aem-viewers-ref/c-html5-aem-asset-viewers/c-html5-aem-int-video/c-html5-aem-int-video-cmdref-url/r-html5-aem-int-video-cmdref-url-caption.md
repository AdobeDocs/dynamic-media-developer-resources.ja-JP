# caption{#caption}

インタラクティブビデオビューアのURLコマンド

` caption= *`ファイル`*[,0|1]`

ビューアでは、ホストされているWebVTTファイルを使用したクローズドキャプションがサポートされます。 キューと領域の重なりはサポートされていません。 サポートされるキュー位置の演算子は次のとおりです。

# callToActionRecap{#calltoactionrecap}

インタラクティブビデオビューアの設定属性。

`callToActionRecap=0|1`

 <tbody> 
  <tr> 
   <td colname="col1"> <p> <span class="codeph"> 0 | 1</span> </p> </td>

  </tr> 
 </tbody>

# インタラクティブ画像ビューアのJavaScript APIリファレンス{#javascript-api-reference-for-interactive-image-viewer}

インタラクティブ画像ビューアのメインクラスはInteractiveImageです。 s7viewers名前空間で宣言されています。 このJavaScript APIは、このクラスのコンストラクター、メソッドおよびコールバックをカバーします。

この後に示す例で、は、ク `<instance>` ラスからインスタンス化された、JavaScriptビューアオブジェクトの実際の名前 `s7viewers.InteractiveImage` です。

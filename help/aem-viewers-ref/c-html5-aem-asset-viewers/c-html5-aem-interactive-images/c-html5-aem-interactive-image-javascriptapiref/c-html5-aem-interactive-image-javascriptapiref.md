# インタラクティブ画像ビューアのJavaScript APIリファレンス{#javascript-api-reference-for-interactive-image-viewer}

この後に示す例で、 `<instance>`は、`s7viewers.InteractiveImage`クラスからインスタンス化された、JavaScriptビューアオブジェクトの実際の名前です。

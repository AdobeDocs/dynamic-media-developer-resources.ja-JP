# キーボードのアクセシビリティとナビゲーション{#keyboard-accessibility-and-navigation}

Viewer UI で公開されたすべての機能がキーボードでアクセスできる。
エンドユーザーは `Left`, `Right`, `UP` および `DOWN` キーを押すと、画像がパンされます。
この `Esc` キーストロークを使用して画像をリセットできます。

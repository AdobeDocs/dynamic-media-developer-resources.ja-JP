# キーボードアクセシビリティとナビゲーション{#keyboard-accessibility-and-navigation}

ビューアのユーザーインターフェイスに表示されるすべての機能は、キーボードでアクセスできます。
エンドユーザーは、`Left`、`Right`、`UP` および `DOWN` のキーストロークを使用して画像をパンできます。
`Esc` のキーストロークを使用して、画像をリセットできます。

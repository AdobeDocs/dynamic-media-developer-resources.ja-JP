# SpinView.zoomstep{#spinview-zoomstep}

` [SpinView.|<containerId>_spinView.]zoomstep= *``*[, *`脚`*]`

   <td colname="col1"> <p> <span class="codeph"><span class="varname"> 手順</span></span> </p> </td>

  </tr>

# 基本ズームビューアのJavaScript APIリファレンス{#javascript-api-reference-for-basic-zoom-viewer}

この後に示す例で、は、ク `<instance>` ラスからインスタンス化された、JavaScriptビューアオブジェクトの実際の名前 `s7viewers.BasicZoomViewer` です。

# ZoomView.reset{#zoomview-reset}

 <tbody> 
  <tr> 
   <td colname="col1"> <p> <span class="codeph"> 0 | 1</span> </p> </td> 
   <td colname="col2"> <p> フレーム（画像）が変更された場合にビューポートをリセットします。 0に設定すると <span class="varname"></span> 、新しく設定した画像の縦横比を保ちつつ、現在のビューポートができる限り適切に合うように保持されます。 </p> </td>

# 検索機能{#search-feature}

ビューアは、カタログのコンテンツに対するテキスト検索をサポートします。 検索機能が機能するには、ソース PDF ファイルをDynamic Media Classicにアップロードするときに、「検索語を抽出」機能をオンにする必要があります。

検索機能は、メインツールバーの「検索」ボタンをアクティブにすることでトリガーされます。 このアクションは、テキスト入力フィールドを含む検索結果パネルを表示します。 ユーザーは、スペースで区切られたキーワードを使用するか、二重引用符で囲まれたフレーズを使用して検索を実行できます。

検索結果は、検索テキストが見つかったページのサムネールとして表示されます。 さらに、ビューアには、検索中に見つかったページとヒットの合計数が表示されます。 ユーザーは、検索結果のサムネールをクリックまたはタップして、カタログの対応するページに移動できます。

最後に、半透明のイエロー領域を使用して、メインビューの検索ヒットをハイライト表示します。

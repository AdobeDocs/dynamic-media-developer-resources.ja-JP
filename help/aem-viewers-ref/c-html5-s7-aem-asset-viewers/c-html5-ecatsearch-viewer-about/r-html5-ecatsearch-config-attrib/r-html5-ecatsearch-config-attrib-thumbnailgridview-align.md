   <td> <p> コンポーネント領域内のサムネールコンテナの内部水平方向の整列（アンカー）を指定します。 ThumbnailGridViewでは、内部のサムネールコンテナのサイズが調整され、サムネールが整数だけ表示されます。 その結果、内部コンテナと外部コンポーネントの境界の間にパディングが生じます。 この修飾子は、内部サムネールコンテナをコンポーネント内で水平方向に配置する方法を指定します。 </p> </td>

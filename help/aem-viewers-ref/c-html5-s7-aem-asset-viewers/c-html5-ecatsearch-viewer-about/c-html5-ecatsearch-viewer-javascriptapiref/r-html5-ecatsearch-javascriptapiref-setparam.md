# setParam{#setparam}

eCatalogビューアのJavaScript APIリファレンス。

[!DNL ` setParam( *`名前、値`*)`]

ビューアのパラメータを指定した値に設定します。 パラメータは、ビューア固有の設定オプションまたはソフトウェア開発キット修飾子です。 このパラメータは[!DNL `init()`]の前に呼び出されます。

ビューアの設定情報が[!DNL `config`] JSONオブジェクトを使用して渡される場合は、このメソッドはオプションです。

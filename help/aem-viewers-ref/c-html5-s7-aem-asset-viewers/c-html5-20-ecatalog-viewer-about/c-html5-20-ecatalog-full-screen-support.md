# フルスクリーンサポート{#full-screen-support}

ビューアはフルスクリーン操作モードをサポートしています。

Internet Explorer 10以前を除く最新のデスクトップブラウザでは、一部のタッチデバイスでは、ビューアは「ネイティブ」フルスクリーンモードを使用します。 このモードは、デバイス画面全体がビューアコンテンツによって占められることを意味します。

IOS デバイスと古いInternet Explorer ブラウザーでは、ビューアは代わりに「シミュレートされた」フルスクリーンモードを使用します。 このモードでは、ビューアのサイズを変更するだけで、web ブラウザーウィンドウの全領域が表示されます。 また、web ブラウザーのユーザーインターフェイスやその他のウィンドウは、引き続き画面に表示されます。

エンドユーザーは、ビューアユーザーインターフェイスのフルスクリーンボタンを押すと、フルスクリーンモードに入り、フルスクリーンモードから離れます。 デスクトップで「ネイティブ」フルスクリーンモードを使用している場合は、**Esc**&#x200B;を押して終了することもできます。

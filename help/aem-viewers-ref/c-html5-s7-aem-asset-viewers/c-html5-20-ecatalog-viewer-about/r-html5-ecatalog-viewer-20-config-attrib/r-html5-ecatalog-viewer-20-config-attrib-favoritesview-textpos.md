   <td colname="col2"> <p> サムネール画像を基準にしてラベルを描画する場所を指定します。 つまり、ラベルはサムネールに対して指定された場所に中央に配置されます。 </p> <p><span class="codeph"> ツールチップ </span>を指定すると、ラベルテキストがサムネール画像の上にフローティングツールチップとして表示されます。 </p> <p><span class="codeph"> none</span>に設定すると、ラベル表示がオフになります。 </p> </td>

# eCatalogビューアのJavaScript APIリファレンス{#javascript-api-reference-for-ecatalog-viewer}

この後に示す例で、 `<instance>`は、`s7viewers.eCatalogViewer`クラスからインスタンス化された、JavaScriptビューアオブジェクトの実際の名前です。

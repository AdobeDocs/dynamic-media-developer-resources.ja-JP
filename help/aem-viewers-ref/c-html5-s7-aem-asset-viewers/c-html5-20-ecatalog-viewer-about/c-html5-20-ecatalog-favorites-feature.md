# お気に入り機能{#favorites-feature}

エンドユーザーは、「お気に入り」機能を使用して、電子カタログ画像に「ハート」アイコンを追加することで、電子カタログ内で関心のある製品を直接マークできます。 ビューアは、クライアントのシステム上にあるすべてのお気に入りを保存し、同じユーザーが電子カタログに再びアクセスしたときに再度表示します。

さらに、ビューアでは、サムネール画像のグリッドとして、電子カタログに追加されたすべてのお気に入りを確認できます。

お気に入りツールはすべて、コントロールバーの1つのドロップダウンリストにグループ化されます。

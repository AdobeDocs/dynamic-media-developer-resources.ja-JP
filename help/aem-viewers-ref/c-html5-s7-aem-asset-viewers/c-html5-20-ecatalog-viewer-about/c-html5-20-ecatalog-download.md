# ダウンロード:{#download}

コントロールバーの「ダウンロード」ボタンを使用して、電子カタログをPDFファイルとしてダウンロードできます。

次の場合、Viewerのユーザインターフェイスで「ダウンロード」ボタンが自動的に使用可能になります。

* 実際のPDFファイルは、顧客の会社に存在します。
* この名前は、ビューアに渡され、拡張子を含むe-catalogアセットの名前と一致 `.pdf` します。
* PDFファイルはSPSで公開されます。

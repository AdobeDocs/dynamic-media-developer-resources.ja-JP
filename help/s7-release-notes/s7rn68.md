## リリースハイライト {#section-524cffc79c1f4bb38627a5ecccd0bf52}

* レスポンシブ web デザインのサポートが追加されました。ブレークポイントに基づくレスポンシブ画像サービングと、ページレイアウトに基づいて動的にサイズ変更されるレスポンシブ HTML5 ビューア（画像セット、ビデオ、混在メディア、eCatalog、スピンセット、Swatchsets、ズーム）です。
* Adobe Analytics ビデオレポートの統合。
* 高度なHTML5 ビューアのカスタマイズ – 埋め込み web ページを使用したカスタムビューアインタラクションを作成します。
* 画像機能強化 – 配信時に、埋め込みパスに画像を切り抜く機能を追加しました。

## ビューア {#section-609885a9c8514a4095da74a0b526138d}

次の更新がおこなわれました。

* Windows オペレーティングシステムの最小システム要件は、Windows 7 になりました。
* Internet Explorer の最小システム要件は、バージョン 9 になりました。
* macOSの最小システム要件は、macOS 10.8 になりました。
* Safari の最小システム要件は、バージョン 6 になりました。

リファクタリングされたものを次に示します。

* レスポンシブデザインをサポートするHTML5 ビューア。
* HTML5 ビューア：CSS 入力およびサイズマーカーを使用して、デスクトップおよびモバイルデバイスでの表示を制御します。

次の機能がサポートされるようになりました。

* Adobe Analytics 5 ビデオビューアおよび混在メディアビューア用のHTML ビデオレポートの統合。
* 同じページ上の複数のHTML5 ビューアのイベントトラッキング。
* HTML5 ズームビューアおよびHTML5 混在メディアビューアに `SetIndicator` を追加して、モバイルデバイス上でスウォッチを視覚的に表すようにしました。
* HTML5 ビューアコンポーネントへの外部アクセスを可能にする `getComponent` API を追加しました。
* 単一の JSON オブジェクトとして渡されるHTML5 ビューア設定パラメーター。
* WebVTT キャプションを JSON オブジェクトとして渡すことで、ルールセットを設定する必要がなくなります。
* 画像マップを外部ページにリンクするためのHTML5 eCatalog ビューアのJavaScript テンプレート。
* HTML5 eCatalog ビューア内のページへのリンク。
* HTML5 Flyout ビューアに `stagesize` 動します。
* ビューアの後で関数をトリガーするためのHTML5 ビューアへの `initComplete` ハンドラー。
* サードパーティのトラッキングをサポートするHTML5 ビューア用の Analytics ハンドラー。
* ソーシャル共有用のリンクと埋め込みコードを簡単に取得できる「すべてを選択」ボタンを追加しました。
* FlyoutZoomView のサイズ変更を追加しました。
* レスポンシブデザインで使用する場合の FlyoutZoomView の画像読み込みブレークポイントの設定。
* 音量を調整するためのクリックサポートを追加しました。
* SpinView および ZoomView：ビューアとやり取りしない場合に、埋め込み Web ページにジェスチャを適用します。
* 実行時 `unload()` コンポーネントを配置できるように、SpinView と ZoomView に API を追加しました。
* 埋め込み Web ページがビューアと対話していないときにジェスチャを適用するビデオビューア。
* ZoomView の初期ズーム領域を設定しています。
* ズームできない場合にコンポーネント `IconEffect` 非表示にする ZoomView

[ ビューアリファレンスガイド ](https://experienceleague.adobe.com/docs/dynamic-media-developer-resources/library/homeviewers.html?lang=en) も参照してください。

## Platform {#section-7e2e7bd11b81424e8cd939b6675ca729}

* 取り込み前に、Clam AntiVirus を使用して受信ファイルのウイルスがチェックされるようになりました。 失敗したファイルは直ちに強制隔離されます。
* ジョブの詳細ポップアップを拡張して、各詳細レコードの時間を含めます。
* アセットからメタデータ値を削除する API を実装しました（deleteAssetMetadata）。
* デフォルトの検索動作を dc.subject を含むように変更しました。
* DOS を避けるために、パスワードリセットメカニズム（パスワードを忘れた場合）を複数ステップのプロセスに変更しました。
* ブロックリストへの登録を防ぐには、パスワードを忘れた場合の E メールフォームのHTMLをプレーンテキストに変換します。
* デフォルトの役割をなくしました。
* サーバーディレクトリのアップロード（ftp）と URL のアップロードを介して Scene7 公開システムにコンテンツをアップロードする機能に加えて、HTTP POST を使用してコンテンツをアップロードできるようになりました。 HTTP POST の使用について詳しくは、[Image Production System API](/help/aem-ips-api/c-overview.md) を参照してください。

## バグの修正 {#section-456515477c8a4aa69309de43292ce5ed}

**ビューア**

* Flash Fastzoom および Flyout ビューアの潜在的なセキュリティの問題。
* 認識できない UserAgent が原因でHTML5 ビューアが読み込まれない。
* 画像マップを含んだページを使用している Internet Explorer で、HTML5 eCatalog ビューアのマウスのドラッグ&amp;ドロップが機能しない。
* HTML5 ビデオビューアは、Flash プラグインのブロックを解除するためのアクセスを妨げます。
* HTML5 ビューアでは、Firefox ブラウザーでの印刷が妨げられます。
* HTML5 ズームビューアでズームを開始またはリセットすると、わずかな揺れが表示される。
* HTML5 および AS3 ビューアの HTTPS サポート。
* autoResize は、HTML Flyout ビューアの URL のコピー/埋め込みコードにしないでください。
* HTML5 ビデオビューアのコピー URL に videoserverurl 修飾子がありません。
* パラメーター `contenturl` は、パブリッシュサーバーのHTML5 ビューアのプレビュー URL から取得されます。

**イメージング**

* 画像公開設定で調整済みビューをデフォルト画像として追加できない。
* colorManagementOptions ラベル「SRGB に変換（デフォルト）」を「デフォルトカラーマネジメント」に変更します。
* ファイル /情報を編集/複数の画像アセットを選択すると、情報を編集できませんでした。

**プラットフォーム**

* Adobe Analytics設定を編集した後、保存せずに設定からエスケープしようとすると、アラートメッセージが追加されました。
* スケジュールされたアップロードジョブを編集すると、プロジェクトオプション参照が「なし」にリセットされる。
* ユーザーの役割に対して File-Move フォルダーを表示しないでください。
* 詳細表示で一部のタブが表示されませんでした。
* Flashplayer のデバッグバージョンがインストールされている場合の Web-to-print 編集エラー。
* 「次の後のスペース」は機能しません。`sa` を使用する必要があります。現在は `margb` です。
* 会社設定を初期化する際に、Scene7 公開システムが API `<defaultValue>` で重複するオプション `<createMetadataField>` を送信します。

* 十分な量の Flash メモリがないフォルダーが多いため、Flash プラグインと Scene7 パブリッシュシステムがクラッシュしました。
* 「日付/時間」ユーザー定義フィールドの値を削除すると、Scene7 公開システムが IPS （Image Publishing System）にリクエストを送信しません。
* Evideo エンコーディングオプションが有効な場合、PDFまたはPSD ファイルの再処理でアダプティブビデオエンコーディングの警告メッセージが発生しました。
* SSL が有効な場合、アセットの書き出しに失敗します。
* パスワードを忘れた場合のメールに、誤ったパスワードが表示されていました。
* Scene7 公開システムの web 管理リクエストは、HTTP にリダイレクトされます。
* プリセットが選択されていない場合、アップロード中にすべてのバッチセットプリセットが適用されていました。
* Scene7 公開システムは、s7info コンテンツのアップロード時に、HTTP を介して s7info をリクエストします。
* パスワードを忘れた場合のメールの URL は、SSL Scene7 公開システムから送信された URL であっても、HTTP です。
* 特殊文字がHTML形式の「パスワードを忘れる」メールにエンコードされ、パスワードが正しく表示されない問題を修正しました。
* Scene7 FTP アカウント情報が体験版アカウントのようこそメールにない
* プロジェクトオプションと切り抜きオプションの両方が選択されている場合、再処理ジョブが失敗しました。
* ジョブオプションが会社のデフォルトにリセットされた場合、アップロードファイルが機能しない。
* Adobe Analyticsの設定ページに「会社」というラベルが完全には表示されません。
* メタデータを読み込めません。

## 既知の問題 {#section-b2608177928b4174b080d89c938bb0d3}

* 2014 年 5 月のAdobe IllustratorとAdobe InDesignのアップデートでは、Scene7 FXG を作成するプラグインとの非互換性が生じる予定です。 これらの非互換性は、Scene7 web/プリントソリューションのユーザーに影響を与えます。 Scene7 Web から印刷を使用する場合は、IllustratorとInDesignをこのCreative Cloudのアップデートにアップグレードしないでください。 代わりに、2014 年 5 月より前のバージョンを引き続き使用してください。 Adobe Inc では、Scene7 FXG 作成とIllustratorおよびInDesignの新しいバージョンとの互換性が、2014 年秋の期間で回復する予定です。
* ユニバーサルビューアプリセットに無効なデフォルトの AS2 プリセットが含まれている場合、埋め込みコードボタンが機能しない。
* アダプティブビデオセットの再処理では、常に新しいアダプティブビデオセットが作成されます。
* [ ジョブの詳細 ] ダイアログ ボックスのユーザーインターフェイスが、詳細に時間を追加した直後に切り捨てられます。

## サポートされなくなった機能 {#section-7b04b6fdb8d1473880ccaf84211254ad}

* テキストハイフネーションは、Scene7 6.9 の秋リリースでサポートされなくなりました。 ハイフネーションの辞書では、最適なテキストフローをサポートするために、単語を分割したりハイフンを付けたりする方法を定義します。
* 2014 年 1 月 31 日（PT）をもって、Scene7 は DHTML ビューアプラットフォームのサポートを正式に終了しました。
* このリリースでは、ダイナミック（フラッシュ）バナーは非推奨（廃止予定）になっています。 パーソナライズされた web サイトバナーの作成、管理および配信に関する他のテクノロジーが登場するようになったので、この機能の採用は少なくなっています。 Scene7 Dynamic （Flash）バナー機能は、サーバーサイドでは引き続き機能しますが、サポートされなくなりました。
* 注意：以前の Scene7 リリースの時点で、メタデータサーバーと関連 API の使用が正式にサポートされていません。
* Scene7 Creative Suite 拡張機能（フラッシュパネル）は 2014 年末に非推奨（廃止予定）になりました。 コンテンツ取得ワークフローのさらなる開発では、Adobe Experience Manager Dynamic Media とCreative Cloudに重点を置いています。 Scene7 の場合、Scene7 パブリッシュシステムのアップロード機能とエクスポート機能は引き続き使用でき、代わりに使用できます。
* 2014 年 5 月をもって、Scene7 でのキャッシュの無効化が完了した時点で、Akamai はメール通知をサポートしなくなりました。 CDN 無効化の一般的な処理時間は約 8 分です。 リクエストのキューが大きい場合、または URL が多数ある場合は、処理時間が若干長くなる可能性があります。
* 詳細表示の「URL と埋め込みコード」タブの「ユニバーサルプリセット」カテゴリ。
* ビデオの詳細表示用 RTMP ストリーミング。
* Scene7 公開システムの「メール通信」および SendMailSimple.jsp サーブレット
* メディアポータルユーザー向けのようこそメールテンプレートの「Scene7 FTP アカウント」セクション
* ips （Image Publishing System）インスタンスがダウンする原因となったので、フィルターパネルから dc.title を選択します。
* HTML5 ビューアでフルスクリーンモードで表示されるソーシャル機能。
* HTML5 ビューアでフルスクリーンモードで表示されている「閉じる」ボタン。

## S7OnDemand 5.0.1 の新機能とバグ修正 {#section-ba3a595134144711bf27dd8dca249e5d}

* 修正：HTML5 および AS3 ビューアの HTTPS サポート。
* HTML5 ビューアの埋め込みコードにレスポンシブサポートを追加しました。
* HTML5 ビューアの埋め込みコードは、設定パラメーターを単一の JSON オブジェクトとして渡します。

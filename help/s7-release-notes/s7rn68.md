## リリースハイライト {#section-524cffc79c1f4bb38627a5ecccd0bf52}

* レスポンシブ web デザインのサポートが追加されました。ページレイアウトに基づいて動的にサイズ変更される、ブレイクポイントおよびレスポンシブ HTML5 ビューア（画像セット、ビデオ、Mixed Media、eCatalog、スピンセット、スウォッチセット、ズーム）に基づくレスポンシブ画像の配信が追加されました。
* Adobe Analytics Video Reportの統合：
* 高度なHTML5 ビューアのカスタマイズ – 埋め込みweb ページを使用してカスタムビューアインタラクションを作成します。
* 画像の強化 – 配信時に埋め込まれたパスに画像を切り抜く機能を追加しました。

## ビューア {#section-609885a9c8514a4095da74a0b526138d}

以下が更新されました。

* Windows オペレーティングシステムの最小必要システム構成はWindows 7です。
* Internet Explorerの最小必要システム構成はバージョン 9です。
* MacOSの最小必要システム構成は、macOS 10.8です。
* Safariの最小必要システム構成はバージョン 6です。

以下がリファクタリングされました。

* レスポンシブデザインをサポートするHTML 5 ビューア。
* HTML5 ビューアでは、CSS入力マーカーとサイズマーカーを使用して、デスクトップおよびモバイルデバイスでの表示を制御します。

次の機能がサポートされるようになりました。

* Adobe Analytics Videoは、HTML 5 Video ViewerとMixed Media Viewerの統合を報告します。
* 同じページの複数のHTML5 ビューアのイベントトラッキング。
* HTML5 Zoom ViewerおよびHTML5 Mixed Media Viewerに`SetIndicator`を追加し、モバイルデバイスでスウォッチを視覚的に表現しました。
* HTML5 ビューアコンポーネントへの外部アクセスを許可する`getComponent` APIを追加しました。
* 1つのJSON オブジェクトとして渡されるHTML5 ビューア設定パラメーター。
* WebVTT キャプションをJSON オブジェクトとして渡すことで、ルールセット設定が不要になります。
* 外部ページに画像マップをリンクするためのHTML5 eCatalog ViewerのJavaScript テンプレート。
* HTML5 e カタログビューア内のページへのリンク。
* `stagesize`からHTML5 Flyout Viewerへ。
* ビューアの後に関数をトリガーするためのHTML5 ビューアへの`initComplete` ハンドラー。
* サードパーティのトラッキングをサポートするためのHTML5 ビューアへのAnalytics ハンドラー。
* ソーシャル共有用のリンクと埋め込みコードを簡単に取得できるように、「すべて選択」ボタンを追加しました。
* FlyoutZoomViewのサイズを変更しました。
* レスポンシブデザインでFlyoutZoomViewの画像リロードブレークポイントを設定する。
* ボリュームを調整するためのクリックサポートを追加しました。
* SpinViewとZoomViewを使用して、ビューアを操作しない場合にweb ページを埋め込むときにジェスチャーを適用します。
* `unload()` APIをSpinViewおよびZoomViewに追加し、実行時にコンポーネントを破棄できるようにしました。
* ビューアと対話しない場合に、埋め込みweb ページにジェスチャーを適用するビデオビューア。
* ZoomViewの初期ズーム領域を設定します。
* ズームできない場合に`IconEffect` コンポーネントを非表示にするZoomView

[&#x200B; ビューアリファレンスガイド &#x200B;](https://experienceleague.adobe.com/docs/dynamic-media-developer-resources/library/homeviewers.html?lang=ja)も参照してください。

## Platform {#section-7e2e7bd11b81424e8cd939b6675ca729}

* 取り込み前に、Clam AntiVirusを使用して受信ファイルがウイルスをチェックするようになりました。 失敗したファイルは直ちに強制隔離されます。
* ジョブの詳細ポップアップを拡張して、各詳細レコードの時間を含めます。
* アセットからメタデータ値を削除するAPIを実装しました（deleteAssetMetadata）。
* デフォルトの検索動作をdc.subjectを含めるように変更しました。
* パスワードリセット機構を変更し、DOSを避けるためにマルチステップのプロセスにパスワードを忘れた。
* パスワードのブロックリストに加えるを防ぐには、HTMLのパスワードを忘れた電子メールをプレーンテキストに変換します。
* デフォルトの役割を削除しました。
* サーバーディレクトリアップロード（ftp）とアップロード URLを介してコンテンツをScene7公開システムにアップロードする機能に加えて、HTTP POSTを使用してコンテンツをアップロードできるようになりました。 HTTP POSTの使用について詳しくは、[Image Production System API](/help/aem-ips-api/c-overview.md)を参照してください。

## バグ修正 {#section-456515477c8a4aa69309de43292ce5ed}

**閲覧者**

* Flash FastzoomおよびFlyout ビューアのセキュリティ問題が発生する可能性があります。
* 認識できないUserAgentが原因で、HTML5 ビューアが読み込まれません。
* HTML5 eCatalog Viewerのマウスのドラッグ&amp;ドロップが、画像マップを含むページのInternet Explorerで機能しない。
* HTML5 Video Viewerは、Flash プラグインのブロック解除へのアクセスを禁止します。
* HTML5 ビューアは、Firefox ブラウザーでの印刷を禁止します。
* HTML5 Zoom Viewerは、ズームを開始またはリセットする際に、わずかなジッターを表示します。
* HTML5およびAS3 ビューアのHTTPS サポート。
* autoResizeは、HTML フライアウトビューアコピーのURL/埋め込みコードには含まれません。
* HTML5 ビデオビューアのコピーURLにvideoserverurl修飾子がありません。
* パラメーター`contenturl`は、パブリッシュサーバーのHTML5 ビューアーのプレビューURLから取得されています。

**イメージング**

* 画像の公開設定で、調整済みビューをデフォルトの画像として追加できません。
* colorManagementOptions ラベルを「SRGBに変換（デフォルト）」から「デフォルトのカラーマネジメント」に変更します。
* ファイル/情報を編集/複数の画像アセットを選択した場合、情報を編集できませんでした。

**Platform**

* Adobe Analytics設定を編集し、保存せずに設定からエスケープしようとすると、警告メッセージが追加されました。
* スケジュールされたアップロードジョブを編集すると、プロジェクトオプションの参照が「なし」にリセットされます。
* ユーザーの役割にファイル移動フォルダーを表示しないでください。
* 詳細ビューに一部のタブが表示されませんでした。
* Flashplayerのデバッグバージョンがインストールされている場合のWebから印刷への編集エラー。
* 「次のスペース」は機能しません。`sa`を使用してください。現在は`margb`です。
* 会社設定を初期化する際、Scene7 パブリッシングシステムは、API `<createMetadataField>`に重複するオプション `<defaultValue>`を送信します。

* Flash プラグインとScene7 パブリッシングシステムが、多くのフォルダーに十分なFlash メモリがないためにクラッシュしました。
* 日付/時刻ユーザー定義フィールドの値を削除する場合、Scene7 Publishing SystemはIPS （Image Publishing System）にリクエストを送信しません。
* ビデオエンコーディングオプションが有効になっている場合、PDFまたはPSD ファイルを再処理すると、アダプティブビデオエンコーディングの警告メッセージが表示されます。
* SSLが有効になっている場合、アセットの書き出しに失敗します。
* パスワードを忘れた電子メールに、誤ったパスワードが表示されていました。
* Scene7 Publishing System Web Admin リクエストはすべてHTTPにリダイレクトされます。
* プリセットが選択されていない場合、アップロード中にすべてのバッチセットプリセットが適用されていました。
* s7info コンテンツをアップロードする際に、HTTPを介してScene7 パブリッシングシステムがs7infoをリクエストします。
* パスワードを忘れた電子メールのURLは、SSL Scene7 パブリッシングシステムから取得した場合でも、HTTPです。
* HTML形式の「パスワードを忘れる」電子メールに特殊文字がエンコードされていたため、パスワードが正しく表示されませんでした。
* 体験版アカウントのウェルカムメールにScene7 FTP アカウント情報が含まれていない
* プロジェクト オプションと切り抜きオプションの両方が選択されている場合、ジョブの再処理に失敗しました。
* ジョブオプションが会社のデフォルトにリセットされている場合、アップロードファイルが機能しない。
* ラベル会社がAdobe Analyticsの設定ページに完全に表示されない。
* メタデータを読み込めません。

## 既知の問題 {#section-b2608177928b4174b080d89c938bb0d3}

* Adobe IllustratorとAdobe InDesignの2014年5月のアップデートでは、Scene7 FXGを作成するプラグインとの互換性が失われることが予想されます。 これらの非互換性は、Scene7 Web-to-Print ソリューションのユーザーに影響します。 Scene7 Web-to-Printを使用している場合は、IllustratorとInDesignをこのCreative Cloud アップデートにアップグレードしないでください。 代わりに、2014年5月以前のバージョンを引き続き使用してください。 Adobe Inc.では、2014年秋にScene7 FXGの制作と新しいバージョンのIllustratorおよびInDesignとの互換性が回復する予定です。
* ユニバーサルビューアプリセットに無効なデフォルトのAS2 プリセットが含まれている場合、「コードを埋め込む」ボタンが機能しない。
* アダプティブビデオセットを再処理すると、常に新しいアダプティブビデオセットが作成されます。
* 詳細に時間を追加すると、ジョブの詳細ダイアログボックスのユーザーインターフェイスが右側で切り捨てられます。

## サポートされなくなった機能 {#section-7b04b6fdb8d1473880ccaf84211254ad}

* テキストハイフネーションは、Scene7 6.9秋リリースではサポートされていません。 ハイフネーション辞書は、最適なテキストフローをサポートするために、単語を分割およびハイフネーションする方法を定義します。
* 2014年1月31日、Scene7はDHTML ビューアプラットフォームのサポートを正式に終了しました。
* Dynamic （Flash） バナーは、このリリースで非推奨（廃止予定）です。 パーソナライズされたweb サイトバナーを作成、管理、配信するための他のテクノロジーが登場しているため、この機能の導入は少なくなっています。 Scene7 Dynamic （Flash）バナー機能は、サーバーサイドで引き続き機能しますが、サポートされなくなりました。
* 注意事項：メタデータサーバーと関連するAPIの使用は、以前のScene7 リリースで正式にサポートを終了しました。
* Scene7 Creative Suite Extensions （Flash Panels）は2014年末に非推奨（廃止予定）となりました。 コンテンツ取り込みワークフローのさらなる発展は、Adobe Experience Manager Dynamic MediaとCreative Cloudに注力しています。 Scene7の場合、Scene7公開システムのアップロード機能と書き出し機能は残り、代わりに使用できます。
* 2014年5月の時点で、Scene7でのキャッシュ無効化が完了すると、Akamaiはメール通知をサポートしなくなりました。 CDN無効化の一般的な処理時間は、約8分です。 リクエストのキューが大きい場合、またはURLが多い場合は、処理時間がわずかに長くなる可能性があります。
* 「URLと埋め込みコード」タブのカテゴリ「ユニバーサルプリセット」の詳細ビュー。
* ビデオの詳細ビューのRTMP ストリーミング。
* Scene7 Publishing Systemの「Email Communications」とSendMailSimple.jsp サーブレット。
* Media Portal ユーザーのウェルカムメールテンプレートの「Scene7 FTP アカウント」セクション。
* IPS （Image Publishing System）インスタンスがダウンしたため、フィルターパネルのdc.titleを削除します。
* HTML5 ビューアでフルスクリーンモードで表示されるソーシャル機能。
* HTML5 ビューアでフルスクリーンモードで表示される「閉じる」ボタン。

## S7OnDemand 5.0.1の新機能とバグ修正 {#section-ba3a595134144711bf27dd8dca249e5d}

* 修正：HTML5およびAS3 ビューアのHTTPS サポート。
* HTML5 ビューアの埋め込みコードにレスポンシブサポートを追加しました。
* HTML5 ビューアの埋め込みコードは、設定パラメーターを1つのJSON オブジェクトとして渡します。

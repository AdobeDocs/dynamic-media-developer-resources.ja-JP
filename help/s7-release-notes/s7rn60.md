# Scene7Publishing System 6.0リリースノート{#scene-publishing-system-release-notes}

Adobe Marketing Cloudの一部であるAdobe Scene7Publishing System 6.0の機能、拡張機能および修正について説明するリリースノートです。

## 新機能と拡張機能{#section-4bbd330df5a043f7a56eee619ef7c442}

**一般**

* 中国語をサポートしました。
* 以下の管理機能が強化されました。

   * 会社ユーザーのリストを書き出します。
   * ユーザ宛てBCC電子メールの送信
   * ユーザーのパスワードをランダムなパスワードにリセットし、ユーザーが再設定できるようにします。

**アセットの管理と公開**

* アセット公開のワークフローが改善されました。<!-- [More information](http://help.adobe.com/en_US/scene7/using/WS3673AD39-098B-4f08-8A24-CA51261B7366.html). -->
* テンプレートプレビューの使い勝手を改善。 詳しくは、次を参照してください。

<!--   [Configuring default viewers](http://help.adobe.com/en_US/scene7/using/WS98ca2e6790647c06-76b2d0e0135685cd4a8-8000.html)

  [Previewing an asset based on viewer platform type](http://help.adobe.com/en_US/scene7/using/WS98ca2e6790647c06-2ce305113564963202-7fff.html)

  [Previewing an image asset based on its image preset](http://help.adobe.com/en_US/scene7/using/WS98ca2e6790647c06-2ce305113564963202-7ffe.html) -->

* このリリース以降、新しい即時公開機能が徐々に公開されるようになります。 この機能は、現在の「公開用にマーク」機能に代わるものです。 一部のユーザーは、他のユーザーよりも前に新しい機能を表示します。 公開期間が近づくと、SPSドキュメントとUI要素が更新されます。 公開が始まると、詳細を電子メールでお知らせします。

**ビューア**

* FlashAS3およびHTML5ビューアプリセットのツールヒントテキストにローカライゼーションのサポートを追加しました。
* キャプションを単一のeVideoアセットに追加することで、ビデオの提供先をグローバルマーケットに拡張できます。
* 「埋め込みコード」を追加して、ビューアの導入を簡略化しました。

   [!DNL Embed Code]機能を使用すると、選択したアセットタイプのビューアコードをレビューできます。 コードをクリップボードにコピーして、ビューアを配信する Web ページにペーストすることもできます。埋め込みコードダイアログボックスでは、コードを編集することはできません。 .

   [!DNL Embed Code]機能は、次のアセットタイプのいずれかに対して[!DNL Preview: Viewer List]ページまたは[!DNL Detail View]の[!DNL URLs]パネルを開いたときに使用できます。

   * ユニバーサルビューア
   * ズーム
   * eCatalog
   * 画像セット
   * スウォッチセット
   * Spin
   * 再編集ビデオ
   * アダプティブビデオセット
   * ビデオ
   * 混在メディアセット

<!--   [More information](http://help.adobe.com/en_US/scene7/using/WS98ca2e6790647c06-2ce305113564963202-7fff.html) -->

* HTML5 SDKに新しい多数のビューアコンポーネントが加わりました。 （詳しくは、インストーラーに付属の「HTML 5 SDK Readme」を参照）。
* FlashSDKが強化されました。 (詳しくは、インストーラーに付属の「FlashSDKお読みください」を参照してください)。
* 新しいFlashAS3ビューアには、次の機能が含まれます。

   * フライアウトズーム
   * 画像/スウォッチセットのスライドショー
   * 自動スピンを使用した混在メディア
   * 自動スピンによるスピン
   * eCatalog
   * Video_caption

* 新しいHTML5ビューアには、次の機能が含まれます。

   * ズーム
   * 画像セット
   * スウォッチセット

* テキストやグラフィックのローカライゼーションなど、ビューアのグローバル化サポートを追加しました。 SDKインストーラーに付属のFlashガイドまたはHTML5ビューアSDKユーザーガイドを参照してください。
* 埋め込みビデオおよびスピンビューアを使用して、eCatalogにインタラクティブ機能を追加しました。
* Adobe Developer ConnectionからのFlashおよびHTML5ビューアSDKへのアクセスを簡略化しました。<!-- [More information](http://help.adobe.com/en_US/scene7/using/WSd4272150f67705c11b002eec12fcba4dee6-8000.html). -->
* ユニバーサルビューアプロファイルの拡張 — ビューアタイプごとに複数のプロファイルを持つことができるようになりました。<!-- [More information](http://help.adobe.com/en_US/scene7/using/WS1c46793299cf21d73076df86131b02b67e8-8000.html). -->
* デバイス検出を使用して画面解像度やファイルサイズの制約に基づいて動的にサイズを設定する機能を持つ携帯電話の画像配信を改善。<!-- [More information](http://help.adobe.com/en_US/scene7/using/WS1c46793299cf21d7-6ad692c9131d90d137a-8000.html). -->

* Adobe CQ5との統合：アセットの同期、CQ5に埋め込まれたScene7ビューア、Scene7ネットワークを介したCQ5ページでのリッチメディアコンテンツの動的な配信。
* AdobeSearch&amp;Promoteと新しく統合されたワークフロー：Search&amp;Promoteキャンペーンルールに基づいて動的にアセンブルされるScene7のマーケティングバナーを、検索ページとナビゲーションページに簡単に追加できます。

   AdobeSearch&amp;Promoteドキュメントの「バナーについて」を参照してください。

* Scene7/Creative Suite-AdobeCreative Suite6の今後のリリースとの互換性があります。

**Media Portal**

* 大きなファイルの書き出しがタイムアウトになりません。
* ワークグループ(admin)内のユーザー用にFTPアカウントを作成します。
* グループおよびアセット/権限(admin)を持つユーザーのリストを書き出します。

* メタデータスキーマ

   メタデータスキーマは、メタデータプロパティのセットを定義します。 Metadataスキーマの各プロパティには、文字列、浮動小数点、整数、日付、はい/いいえなどの基本的なデータ型があります。 また、リスト、選択肢を開く、選択肢を閉じるなどの定義も可能です。 メタデータスキーマは、検索、編集、読み込み、メタデータ表示やメタデータプリセットなど、すべてのメタデータワークフローで使用できます。<!-- [More information](http://help.adobe.com/en_US/scene7/using/WS259993e42159a215-1c6a66df1265272619e-7fec.html#WSd968ca97bf03cf72-5e3dd3a113268dc80f5-8000). -->

   メタデータスキーマはUDFの後継技術で、特にメタデータサーバのコンテキストでカスタマイズされたメタデータワークフローを対象としています。

* メタデータサーバ

   >[!NOTE]
   >
   >メタデータサーバは5月からご利用いただけます。 サポートが有効にする必要があります。 メタデータサーバは現在、北米およびヨーロッパでのみ使用できます。

* メタデータサーバは、アセットのすべてのメタデータフィールドのサブセットをパブリック側のサーバに公開し、ここでeComページまたはサーバから要求できます。 この公開されたメタデータは、アセットプロパティ、UDF、XMP、メタデータスキーマプロパティの組み合わせセットになる場合があります。 メタデータサーバは、メタデータの検索や画像類似性の検索など、様々な検索機能を備えたWEB-APIをオファーします。 返される各アセットに対して、メタデータプロパティのリストが要求されます。<!-- [More information](http://help.adobe.com/en_US/scene7/using/WS1ffbb36e209a6fc9-44279087131d3ad5622-8000.html). -->
* メタデータの取り込みでXMPプロパティを処理できる

   メタデータの取り込み機能が拡張され、XMPのプロパティとプロパティ(メタデータスキーマで定義)を処理できるようになりました。
* 値を含むフィールドのみを表示するメタデータ表示

   定義済みの新しいメタデータ表示「値を持つすべてのプロパティ」に、値を含むメタプロパティのみが表示されます。 値を持たないプロパティ(XMP、ユーザ定義フィールドまたは新しいメタデータスキーマから取得)は非表示になり、表示されません。
* 管理ユーザーインターフェイスにグループ化されたメタデータ設定

   **[!UICONTROL 設定/アプリケーション設定]**&#x200B;で、メタデータ関連の設定ページが、[!DNL Metadata]という最上位セクションにグループ化されました。 メタデータ表示、メタデータプリセット、メタデータスキーマおよびユーザ定義フィールドの設定ページを見つけます。

このデスクトップアプリケーションを使用すると、既存の3Dモデルを読み込んで、2Dビネットファイルとしてオーサリングできます。 作成したビネットファイルをScene7 Publishing Systemにアップロードし、そこで色とテクスチャをオンデマンドで置き換えることができます。

## ソフトウェア修正{#section-4219d5dd0a024cd3a95aaac45598363c}

メタデータサーバで次の問題が修正されました。

* 非常に大きなEPSファイルのアップロードが失敗する問題を解決しました。
* 長い結果リストを伴う「メタデータ検索」は何も示さずに失敗します。 この問題により、誤った検索結果が表示されます。

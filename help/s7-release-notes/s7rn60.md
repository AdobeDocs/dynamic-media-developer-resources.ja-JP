# Scene7 Publishing System 6.0 リリースノート{#scene-publishing-system-release-notes}

Adobe Marketing CloudのAdobe Scene7 Publishing System 6.0 パートの機能、機能強化、および修正について説明するリリースノートです。

## 新機能と機能強化 {#section-4bbd330df5a043f7a56eee619ef7c442}

**一般**

* 中国語のサポート。
* 次のような管理機能の強化：

   * 会社ユーザーのリストをエクスポートします。
   * ユーザーにBCC メールを送信します。
   * ユーザーのパスワードを、ユーザーがリセットできるランダムなパスワードにリセットします。

**アセット管理と公開**

* アセットの公開ワークフローを改善。<!-- [More information](http://help.adobe.com/en_US/scene7/using/WS3673AD39-098B-4f08-8A24-CA51261B7366.html). -->
* テンプレートのプレビューの操作性が向上しました。 詳しくは、次を参照してください。

* このリリースに続いて、新しいインスタント公開機能が徐々にロールアウトされます。 この機能は、現在の「公開用にマーク」機能に代わるものです。 一部のユーザーは、新しい機能を他のユーザーよりも先に見る場合があります。 SPS ドキュメントとUI要素は、ロールアウト期間が完了に近づいたときに更新するように計画されています。 ロールアウト開始時に、より詳細な情報を記載したメールを受信するように計画することができます。

**閲覧者**

* Flash AS3およびHTML5 ビューアプリセットでのツールヒントのテキストのローカライズをサポートしました。
* 単一のeVideo アセットにキャプションを追加することで、動画のリーチをグローバル市場に拡大します。
* ビューアのデプロイメントを簡素化するために「埋め込みコード」を追加しました。

  [!DNL Embed Code]機能を使用すると、選択したアセットタイプのビューアーコードを確認できます。 コードをクリップボードにコピーして、ビューアを配信する Web ページにペーストすることもできます。 埋め込みコードダイアログボックスでは、コードを編集することはできません。

  [!DNL Embed Code]機能は、次のいずれかのアセットタイプの[!DNL Preview: Viewer List] ページまたは[!DNL Detail View]の[!DNL URLs] パネルを開いたときに利用できます。

   * ユニバーサルビューアー
   * ズーム
   * eCatalog
   * 画像セット
   * スウォッチセット
   * Spin
   * 再編集ビデオ
   * アダプティブビデオセット
   * ビデオ
   * 混在メディアセット

**Media Portal**

* タイムアウトが発生することなく大きなファイルを書き出します。
* ワークグループ（管理者）のユーザーのFTP アカウントを作成します。
* グループとアセット/権限（管理者）を持つユーザーのリストを書き出します。

* メタデータスキーマ

  メタデータスキーマは、メタデータプロパティのセットを定義します。 メタデータスキーマの各プロパティには、文字列、浮動小数、整数、日付またはYes/Noなどの基本的なデータタイプがあります。 また、リスト、開いた選択肢または閉じた選択肢も定義できます。 メタデータスキーマは、検索、編集、読み込み、メタデータビュー、メタデータプリセットなど、あらゆるメタデータワークフローで使用できます。<!-- [More information](http://help.adobe.com/en_US/scene7/using/WS259993e42159a215-1c6a66df1265272619e-7fec.html#WSd968ca97bf03cf72-5e3dd3a113268dc80f5-8000). -->

  メタデータスキーマは、UDFの後継テクノロジーであり、特にメタデータサーバーのコンテキストでカスタマイズされたメタデータワークフローを対象としています。

* メタデータサーバ

  >[!NOTE]
  >
  >メタデータサーバーは5月から利用可能です。 サポートによって有効にする必要があります。 Metadata Serverは現在、北米およびヨーロッパでのみ利用可能です。

* メタデータサーバーは、アセットのすべてのメタデータフィールドのサブセットを公開サーバーに公開し、eCom ページまたはサーバーからリクエストできます。 この公開されたメタデータは、アセットプロパティ、UDF、XMP、メタデータスキーマプロパティの組み合わせである可能性があります。 メタデータサーバーは、メタデータの検索、画像の類似性、またはその両方など、様々な検索機能を備えたWEB APIを提供します。 返される各アセットに対して、メタデータプロパティのリストが要求されます。
* メタデータの読み込みは、XMP プロパティを処理できます

  メタデータの読み込みは、メタデータスキーマで定義されているXMPのプロパティとプロパティを処理するように拡張されます。
* 値を含むフィールドのみを表示するメタデータビュー

  値を含むメタプロパティのみを表示する、「値を持つすべてのプロパティ」と呼ばれる新しい定義済みメタデータビューがあります。 値のないプロパティ（XMP、ユーザー定義フィールドまたは新しいメタデータスキーマから）は非表示になり、表示されません。
* 管理ユーザーインターフェイスにメタデータ設定がグループ化されるようになりました。

  **[!UICONTROL セットアップ/アプリケーション設定]**&#x200B;で、メタデータ関連の設定ページが[!DNL Metadata]という最上位セクションにグループ化されるようになりました。 ここでは、メタデータビュー、メタデータプリセット、メタデータスキーマ、ユーザー定義フィールドの設定ページを見つけることができます。

このデスクトップアプリケーションでは、既存の3D モデルを読み込み、2D ビネットファイルとしてオーサリングできます。 生成されたビネットファイルをSPSにアップロードして、カラーとテクスチャをオンデマンドで置き換えることができます。

## ソフトウェアの修正 {#section-4219d5dd0a024cd3a95aaac45598363c}

メタデータサーバーで次の問題が修正されました。

* 非常に大きなEPS ファイルのアップロードに失敗する問題を解決しました。
* 長い結果リストを持つ「メタデータ検索」はサイレントに失敗します。 この問題により、間違った検索結果が表示されます。

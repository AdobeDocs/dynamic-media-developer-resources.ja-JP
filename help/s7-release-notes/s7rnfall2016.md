* 後でユーザーを再作成できなかったユーザーの削除に関するバグ修正。

* Scene7環境用に OKTA サービスプロバイダーを設定します。
* Viewer Analytics にExperience Cloud組織 ID のサポートが追加されました。
* Scene7 SAML コンシューマーを実装しました。

## Scene7HTML5 Viewer SDK 3.0.2 {#section-30e2392859c442d1aab2766d0f1d1580}

ユーザーガイドは、クライアントインストールのAdobeHTML5 Viewer SDK フォルダーにあります。 コンポーネント API のドキュメントは、クライアントインストールの docs サブフォルダーにあります。

* VideoPlayer - Windows 7 の Internet Explorer 11 でビデオを再生できませんでした。
* 目次 —  `initialframe` では、HTML5 eCatalog ビューアのモバイルデバイスのポートレイトモードに影響しませんでした。

* 一般

   * HLS ストリーミングビデオ再生が、ほとんどのデスクトップシステムのデフォルトのビデオ配信方法として追加されました。 Flash・ベースの HDS ビデオ・ストリーミングは、代替再生オプションとして引き続き利用できます。

   * eCatalog 固有の SDK コンポーネントに、右から左への向きのサポートを追加しました。

* ControlBar

   * ControlBar コンテンツが使用可能な幅に収まらない場合に備えて、オプションでスクロールを追加しました。

* FlyoutzoomView

   * 以下の問題を修正しました。 `tip=0,-1,0` が原因で、範囲外エラーが発生していました。

   * ダブルタップジェスチャを使用すると、ブラウザー側での拡大/縮小が有効な場合にビューアとページがズームされます。

**既知の問題と制限事項**

* すべてのコンポーネント

   * バージョン 2.7.2 以前では、一部のコンポーネントが `insertBefore()` API その結果、他のコンポーネントに対してコンポーネントインスタンスがいつ作成されたかに関係なく、このようなコンポーネントは重ね順の最下部に配置されます。 2.8.1 リリースでは、すべてのコンポーネントが `appendChild()` API が追加されました。つまり、コンポーネントの重ね順がインスタンスの作成順と一致します。

   * 使用 `iscommand` 画像のアルファチャンネル形式を設定する修飾子はサポートされていません。 コンポーネントを使用 `FMT` パラメーターを使用します。
   * CSS 変換プロパティは現在サポートされていません。

* タッチデバイス

   * タッチデバイスでピンチジェスチャを行っても、ズームイベントが生成されない

* コンテナ

   * コンテナ上の境界線、パディング、余白はサポートされていません。 Adobeでは、親 DIV にスタイル要素を追加することをお勧めします。
   * コンテナのサイズを明示的に設定する必要があります。設定しないと、コンポーネントのサイズが正しく設定される場合があります。

* 印刷コンポーネント

   * ブラウザーの制限により、Internet Explorer 9 では、印刷コンポーネントが用紙上のコンテンツを適切に拡大/縮小できない場合があります。

* IconEffect コンポーネント

   * 次の場合、IconEffect は Internet Explorer でスクリプトエラーを生成します。 `autohide` 無効 ( `0`) をクリックします。

* ImageMapEffect コンポーネント

   * 表示コンポーネント上の画像をパンする際に、再配置アイコンで遅延します。

* MediaSet コンポーネント

   * インラインアセットは、URL 上と同じエンコーディングを要求します。

* NavigationView コンポーネント

   * 現在、このコンポーネントはサイズ変更をサポートしていません。

* PageScrubber コンポーネント

   * iPhone 5 で、PageScrubber バブルがテキストに設定されている場合、トラックに沿ってボタンをスライドするとアーティファクトが表示されています。 使用 `-webkit-background-clip: content;` スタイルでは問題が回避されます。

* SpinView コンポーネント

   * 画像の回転中にスワイプジェスチャを行ってデバイスを回転すると、SpinView がフリーズする場合があります。

* スウォッチコンポーネント

   * 範囲外のスウォッチを選択すると、2 つのハイライトが表示されます。
   * での自動スクロール `selectSwatch()` メソッドの動作が正しくありません。

* VideoPlayer

   * シークが 100%に設定され、フォールバックが auto に設定されている場合、ビデオフレームは更新されません。
   * Chrome、Firefox および Internet Explorer ブラウザーの HLS ストリーミングモードで、ビデオシーク中にマクロブロックが発生することがあります。
   * ポスター画像は、Microsoft® Edge ブラウザーに初めて訪問者に対して表示されない場合があります。
   * プログレッシブ再生を使用する場合、Internet Explorer 9 でのビデオの読み込み後にポスター画像が非表示になることがあります。

## Dynamic Media Image Serving 6.3.2 および画像レンダリング 6.3.2 {#section-19a3e96f52c74757bcdea0f8a11001f2}

* IC ユーティリティ — `downsample2x2` フラグはサポートされなくなりました。 このフラグは、IPS では使用されなくなった 2x2 ダウンサンプラーの品質が低かった。

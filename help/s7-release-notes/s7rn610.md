## Scene7 IPS （画像制作システム） 4.5.0 {#section-3e37ed37b6d146419f4578bd7ffeaa74}

**新機能**

* JDK 1.8を実行しています。
* EXIF方向タグのサポート。
* ディレクトリファイルのアップロード一時FTP ファイルのスキップ（ProFTPで作成）。
* 画像レンダリングでインスタント公開がサポートされるようになりました。 この機能により、レンダリングされたビネットアセットは、公開ジョブを開始することなく、web サイトですぐに起動できるようになります。 この機能を使用したい場合は、テクニカルサポートに連絡して、アカウントの即時公開を有効にしてもらってください。

**機能はサポートされなくなりました**

* IPS Classic Publish Validate内のCatalog Server Resync オプションが削除されました。
* 空のメタデータタグフィールドまたは重複するメタデータタグフィールド値を設定できなくなりました。

**バグ修正**

* API `getAssets`に応答して、[!DNL .vtt] ファイルの`<metadataArray>`が空でした。

* 方向のユーザーデータフィールドがありません。
* ACO ファイルを更新するジョブに対する`getJobLogDetails`からの応答で、ジョブの詳細が正しく指定されていませんでした。
* PSD プライマリファイルは、Image Serverごとに複数回公開されたとして記録されました。
* 名前に無効な文字が含まれているAssetsをIPSにアップロードできます。
* 生成に失敗したビデオサムネールは、「キューで保留中」状態のままになりました。
* 大きな`deleteFolder`とサーバーディレクトリのアップロードの間にフォルダーを復元する競合条件。
* アップロード時に早期に公開すると、PDF ファイルはパブリッシュサーバーごとに2回公開されたとしてログに記録されていました。
* 同じレイヤー名を持つ複数のPSD ファイルをアップロードすると、最適化エラーが発生しました。
* お客様のPDFをPDFLでラスタライズすると、生成された画像に予期しない白い線が生じました。
* 動画のサムネールファイルはアップロード時に複数回公開されました。
* 重複アップロードファイル検出により、2つのアップロード通知が生成され、完了したファイルアカウンティングが破棄されました。
* ビデオトランスコード用のSidecar XML ファイルは、更新時に複数回「公開」として記録されました。
* ネストされたフォルダーを削除し、ネストされたフォルダーの親フォルダーではなく、ネストされたフォルダーと同じ名前を持つ別のフォルダーを、ネストされたフォルダーが以前あった場所に移動したときにmoveFolderが失敗しました。
* 共有およびローカル設定のクラスターステータス設定エントリは、クリーンアップによってパージされませんでした。
* API ログパーサーを変更して、メールアドレス形式をフィルタリングします。
* ビデオ処理で`Iptc4xmpCore:Location`が正しくマッピングされません。
* クリーンアップとアップロードの間に競合状態が存在しました。
* 公開では`assetFile`は使用されないため、ステージング公開は新しいアセットに対してのみ機能します。

## サポートされなくなった機能 {#section-9295ea9313c84695b19bca0e0f01a690}

* **Flash ビューアの提供終了のお知らせ** *2017年1月31日より、Adobe Scene7 （現在のDynamic Media Classic）はFlash ビューアプラットフォームのサポートを正式に終了しました。*

* 注意：2014年1月31日をもって、Scene7はDHTML ビューアプラットフォームのサポートを正式に終了しました。
* Scene7 Creative Suite Extensionsは、Creative Cloud アプリケーションのFlash パネルのサポートが終了したため、このリリースでは非推奨になりました。 コンテンツ取り込みワークフローのさらなる発展は、Adobe Experience Manager Dynamic MediaとCreative Cloudに注力しています。 Scene7の場合、Scene7公開システムのアップロード機能と書き出し機能は残り、代わりに使用できます。
* UGC ビデオは、採用率が低いため、このリリースでは非推奨です。 別のDynamic Media Classic アカウントを設定すると、UGC ビデオの代替ソリューションとして機能する可能性があります。 別のアカウントを設定するには、Adobe テクニカルサポートにお問い合わせください。
* このリリースでは、採用率が低いため、テキストハイフネーションは推奨されません。 ハイフネーション辞書は、最適なテキストフローをサポートするために、単語を分割およびハイフネーションする方法を定義しました。
* メタデータサーバーと類似性検索は、採用率が低いため、2015年に廃止されました。 正式な廃止予定日が発表されると、影響を受けるお客様にはAdobe テクニカルサポートから通知が送られました。

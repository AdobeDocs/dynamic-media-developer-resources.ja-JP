## Scene7 IPS(Image Production System)4.5.0 {#section-3e37ed37b6d146419f4578bd7ffeaa74}

**新機能**

* JDK 1.8を使用した実行
* EXIF方向タグのサポート。
* 一時FTPファイルのアップロードディレクトリファイルのスキップ（ProFTPで作成されたもの）
* 画像レンダリングで即時公開がサポートされるようになりました。 これは、レンダリングされたビネットアセットが、公開ジョブを開始することなく、即座にWebサイト上で起動できることを意味します。 この機能の使用に興味がある場合は、テクニカルサポートに連絡して、アカウントで即時公開を有効にしてもらうように依頼してください。

**サポートされなくなった機能**

* IPS Classic公開検証の「カタログサーバの再同期」オプションが削除されました。
* 空または重複メタデータのタグフィールドの値を設定できなくなりました。

**バグ修正**

* API `<metadataArray>` に応答して、フ [!DNL .vtt] ァイルのが空でした `getAssets`。

* 向きのユーザーデータフィールドがありません。
* ACOファイル `getJobLogDetails` を更新するジョブに対するからの応答で、ジョブの詳細が正しく指定されていませんでした。
* PSDプライマリファイルは、Image Serverごとに複数回公開されたファイルとしてログに記録されました。
* 名前に無効な文字が含まれるアセットは、IPSにアップロードできます。
* 生成に失敗したビデオサムネールが「キューに保留中」の状態のままになっていた問題を修正しました。
* 大きいディレクトリとサーバーディレクトリのアップロードの間でのフォルダー復元に伴う競合状態 `deleteFolder` です。
* アップロード時に早い段階で公開した場合、PDFファイルは公開サーバごとに2回公開済みとしてログに記録されていました。
* 同じレイヤー名を持つ複数のPSDファイルをアップロードすると、最適化エラーが発生する問題を修正しました。
* お客様のPDFをPDFLラスタライズすると、生成される画像に予期しない白い線が表示されていました。
* ビデオサムネールファイルがアップロード時に複数回公開されていた問題を修正しました。
* 重複アップロードファイルの検出で、2つのアップロード通知が生成され、完了したファイルのアカウンティングがスローされます。
* ビデオトランスコードのサイドカーXMLファイルが、更新時に複数回公開されていると記録されていました。
* 入れ子フォルダを削除し、入れ子フォルダの親フォルダ内になく、入れ子フォルダと同じ名前を持つ別のフォルダを、以前の入れ子フォルダの場所に移動したときに、moveFolderが失敗しました。
* 共有構成とローカル構成のクラスター状態構成エントリは、クリーンアップによって削除されませんでした。
* APIログパーサーを変更して、電子メールアドレス形式のフィルターにします。
* ビデオ処理が `Iptc4xmpCore:Location` 正しくマッピングされません。
* クリーンアップとアップロードの間に競合条件が存在しました。
* 公開は使用されないので、ステージング公開 `assetFile` は新しいアセットに対してのみ機能します。
* ルールセットの更新時にNULLポインタがスローされました。

**Scene7画像サービングビューア5.2.2の新機能、拡張機能およびバグ修正**

ビューアのアップグレードは下位互換性があり、既存のWebコードに変更を加える必要はありません。 ただし、弊社のステージング環境で新しいビューアをテストすることをお勧めします。 以下のWebサイトには、弊社のステージングサーバーにアクセスするためのシステムの設定方法が記載されています。

[http://helpx.adobe.com/experience-manager/scene7/kb/base/upgrade-management/testing-scene7-viewers-upgrade.html](http://helpx.adobe.com/experience-manager/scene7/kb/base/upgrade-management/testing-scene7-viewers-upgrade.html)

ステージングサーバーにアクセスするようにコンピューターを設定したら、Webサイトを確認してアップグレードをテストできます。 標準搭載のHTML5ビューアを使用する場合、ベストプラクティスは、弊社の標準のステージングサーバーとs7is5-プレビューステージング.scene7.comに対してテストすることです。

新機能および機能強化

『 [Scene7ビューアリファレンスガイド](https://marketing.adobe.com/resources/help/en_US/s7/viewers_ref/)』も参照してください。

* eCatalogビューアで、印刷、ダウンロード、およびお気に入りのサポートが追加されました。

* コンテナでの（非ネイティブの）フルスクリーンのサポートをリファクタリング。
* より大きなスマートフォンをサポートするために、CSSの小さいマーカーサイズを拡大しました。
* デスクトップブラウザでのHTML5スピン、基本ズーム、ズーム、スピン、およびMixedMediaビューア用のCSSサイズマーカーを削除しました。
* SpinViewでプリロードされたフレームの画質設定を可能にするサポートを追加しました。
* 修正 — Galaxy S4、縦置きモード： フルスクリーンモードが有効な場合に、誤ったCSSサイズマーカーが使用される。
* 修正 — [eCatalog] 画像マップが、Internet Explorer 9での外部リンクにJavaScriptテンプレートを使用した後に停止する問題を修正しました。
* 修正 — Internet Explorer 9およびInternet Explorer 10: CSSのIMGに対してwidthプロパティが定義されている場合、ズームが正しく機能しません。

## Scene7 Publishing System 6.10のバグ修正 {#section-99c1171604ff4ce489285dcab3d929c9}

* 手動で作成したサイズ「1」のスウォッチセットが非公開になっていた問題を修正しました。
* gray pixelTypeのpng8はサポートされません。
* 公開設定で、Scene7の共有アセット会社からのカラープロファイルが表示されませんでした。
* FTP画面からアップロード： FTPフォルダーは並べ替えられませんでしたが、出力フォルダーは並べ替えリストーでした。
* Media Portal設定の「グループを編集」の「アセットのアクセス権限」タブの説明を変更しました。
* 不正な形式 `clientAddressFilter` の場合、セキュアテスト画像サービング公開設定を使用中にカタログサーバから例外が発生していました。
* URLに連続するスペースが複数ある場合に、ChromeブラウザーでPDFのURLをコピーできません。
* Scene7 Publishing SystemのFTPサーバとFTPアカウント情報が一致しない。
* [ [!DNL Adobe Marketing Cloud] リンクウィザード]ボタンが削除されました。
* MPEGビデオがエンコードされていましたが、再生できませんでした。
* 帯域幅の日別レポートが、s7report Webサービスの更新後に機能しなくなりました。
* 接触チャネルダッシュボードドメインレポートで、開始と終了日が正しく設定されていませんでした。
* Scene7 Publishing SystemでのWeb-to-Printのダウンロード操作の絞り込み。
* タグタイプのメタデータフィールドを作成および更新するための設定ユーザーインターフェイスを実装しました。 パラメーターでの **[!UICONTROL メタデータフィールドタイプ]**`fieldType` ( `SingleFixedTag` パラメーター内)のサポートには、を含め `BooleanTag` 、のみが含まれます。

* 「Send email from address」 `@scene7.com` の `@adobe.com` をに変更しました。

* 画像サービングの公開タイプオプションが追加されました。
* 要求本文の日付が正しくないため、ビデオストリーミングデータが表示されませんでした。
* ストレージレポートで履歴ストレージデータを使用できませんでした。

* 帯域幅レポートに、データが表示されているにもかかわらず、帯域幅データが使用できないエラーが発生しました。
* 現在の日付が、選択した日付ではなく、帯域幅とストレージのレポートに表示されていました。
* Scene7 Publishing Systemのようこそページに表示されるメッセージを変更しました。
* アドバンス検索に値 `Userdata` フィールドを追加し直しました。
* タイプのユーザ定義フィールド( **[!UICONTROL 設定]** / **メタデータ** / **ユーザ定義フィールド**)を追加し、その後保存したユーザ定義フィールド `Boolean``true`の値を追加すると、エラーが発生していました。

## サポートされなくなった機能 {#section-9295ea9313c84695b19bca0e0f01a690}

* **Flashビューア提供終了のお知らせ***（2017年1月31日より）Adobe Scene7 Publishing Systemは、Flashビューアプラットフォームのサポートを正式に終了します。*

   *この重要な変更について詳しくは、次のFAQ Webサイトを参照してください。 *

   [https://docs.adobe.com/content/docs/en/aem/6-1/administer/integration/marketing-cloud/scene7/flash-eol.html](https://docs.adobe.com/content/docs/en/aem/6-1/administer/integration/marketing-cloud/scene7/flash-eol.html).

* Reminder: 2014年1月31日をもって、Scene7はDHTMLビューアプラットフォームのサポートを正式に終了しました。

   この変更について詳しくは、次のFAQ Webサイトを参照してください。

   [http://dev.day.com/content/docs/en/cq/current/administering/integrating_with_omniture/scene7/dhtml-viewer-endoflifefaqs.html](http://dev.day.com/content/docs/en/cq/current/administering/integrating_with_omniture/scene7/dhtml-viewer-endoflifefaqs.html)

* Scene7 Creative Suite拡張機能は、Creative CloudアプリケーションによるFlashパネルのサポート廃止により、このリリースで非推奨となりました。 コンテンツ取り込みワークフローのさらなる開発は、Adobe Experience ManagerDynamic MediaとCreative Cloudに重点を置いています。 Scene7では、Scene7 Publishing Systemのアップロード機能と書き出し機能が維持され、代わりに使用できます。
* UGCビデオは、採用が少ないため、このリリースで非推奨となりました。 別のSPSアカウントを設定することが、UGCビデオの代替ソリューションとなります。 アドビテクニカルサポートに問い合わせて、別のアカウントの設定を依頼します。
* テキストハイフネーションは、採用が少ないので、このリリースで非推奨になりました。 ハイフネーション辞書は、最適なテキストフローをサポートするために、単語を分割およびハイフネーションする方法を定義していました。
* メタデータサーバーと類似性検索は、共に採用が少ないため、2015年に廃止されました。 正式な廃止ターゲット日が発表されると、影響を受けるお客様には、アドビテクニカルサポートから通知が送信されます。

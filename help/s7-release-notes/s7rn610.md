## Scene7IPS (Image Production System) 4.5.0 {#section-3e37ed37b6d146419f4578bd7ffeaa74}

**新機能**

* JDK 1.8を使用した実行
* EXIF方向タグのサポート。
* 一時FTPファイルのアップロードディレクトリファイルのスキップ（ProFTPで作成されたもの）
* 画像レンダリングで即時公開がサポートされるようになりました。 これは、レンダリングされたビネットアセットが、公開ジョブを開始することなく、即座にWebサイト上で起動できることを意味します。 この機能の使用に興味がある場合は、テクニカルサポートに連絡して、アカウントで即時公開を有効にしてもらうように依頼してください。

**サポートされなくなった機能**

* IPS Classic公開検証の「カタログサーバの再同期」オプションが削除されました。
* 空または重複メタデータのタグフィールドの値を設定できなくなりました。

**バグ修正**

* API `getAssets`に応答して`<metadataArray>`が[!DNL .vtt]ファイルに対して空でした。

* 向きのユーザーデータフィールドがありません。
* ACOファイルを更新するジョブに対する`getJobLogDetails`からの応答で、ジョブの詳細が正しく指定されていませんでした。
* PSDプライマリファイルは、Image Serverごとに複数回公開されたファイルとしてログに記録されました。
* 名前に無効な文字が含まれるアセットは、IPSにアップロードできます。
* 生成に失敗したビデオサムネールが「キューに保留中」の状態のままになっていた問題を修正しました。
* 大きい`deleteFolder`とサーバーディレクトリのアップロード間でのフォルダー復元と競合状態。
* アップロード時に早い段階で公開した場合、PDFファイルは公開サーバごとに2回公開済みとしてログに記録されていました。
* 同じレイヤー名を持つ複数のPSDファイルをアップロードすると、最適化エラーが発生する問題を修正しました。
* 顧客のPDFをPDFLラスタライズすると、生成される画像に予期しない白い線が表示されていました。
* ビデオサムネールファイルがアップロード時に複数回公開されていた問題を修正しました。
* 重複アップロードファイルの検出で、2つのアップロード通知が生成され、完了したファイルのアカウンティングがスローされます。
* ビデオトランスコードのサイドカーXMLファイルが、更新時に複数回公開されていると記録されていました。
* 入れ子フォルダを削除し、入れ子フォルダの親フォルダ内になく、入れ子フォルダと同じ名前を持つ別のフォルダを、以前の入れ子フォルダの場所に移動したときに、moveFolderが失敗しました。
* 共有構成とローカル構成のクラスター状態構成エントリは、クリーンアップによって削除されませんでした。
* APIログパーサーを変更して、電子メールアドレス形式のフィルターにします。
* ビデオ処理で`Iptc4xmpCore:Location`が正しくマッピングされません。
* クリーンアップとアップロードの間に競合条件が存在しました。
* 公開には`assetFile`が使用されないので、ステージング公開は新しいアセットに対してのみ機能します。
* ルールセットの更新時にNULLポインタがスローされました。

ステージングサーバーにアクセスするようにコンピューターを設定したら、Webサイトを確認してアップグレードをテストできます。 標準搭載のHTML5ビューアを使用する場合、ベストプラクティスは、弊社の標準のステージングサーバーとs7is5-プレビューステージング.scene7.comに対してテストすることです。

## サポートされなくなった機能{#section-9295ea9313c84695b19bca0e0f01a690}

* **Flashビューア提供終了のお** *知らせ2017年1月31日、Adobe Scene7(現在のDynamic Mediaクラシック)はFlashビューアプラットフォームのサポートを正式に終了しました。*

* Reminder:2014年1月31日をもって、Scene7はDHTMLビューアプラットフォームのサポートを正式に終了しました。

   この変更について詳しくは、次のFAQ Webサイトを参照してください。

   [http://dev.day.com/content/docs/en/cq/current/administering/integrating_with_omniture/scene7/dhtml-viewer-endoflifefaqs.html](http://dev.day.com/content/docs/en/cq/current/administering/integrating_with_omniture/scene7/dhtml-viewer-endoflifefaqs.html)

* Scene7Creative Suite拡張機能は、Creative CloudアプリケーションがFlashパネルをサポートしなくなったため、このリリースで非推奨となりました。 コンテンツ取り込みワークフローの更なる開発は、Adobe Experience ManagerDynamic MediaとCreative Cloudに焦点を当てている。 Scene7については、Scene7パブリッシングシステムのアップロード機能と書き出し機能が残り、代わりに使用できます。
* UGCビデオは、採用が少ないため、このリリースで非推奨となりました。 別のDynamic MediaClassicアカウントを設定することが、UGCビデオの代わりのソリューションとなります。 別のアカウントを設定するには、Adobeテクニカルサポートにお問い合わせください。
* テキストハイフネーションは、採用が少ないので、このリリースで非推奨になりました。 ハイフネーション辞書は、最適なテキストフローをサポートするために、単語を分割およびハイフネーションする方法を定義していました。
* メタデータサーバーと類似性検索は、共に採用が少ないため、2015年に廃止されました。 正式な廃止ターゲット日が発表されると、該当するお客様にはAdobeテクニカルサポートから通知が送られます。

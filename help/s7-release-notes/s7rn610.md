## Scene7 IPS(Image Production System)4.5.0 {#section-3e37ed37b6d146419f4578bd7ffeaa74}

**新機能**

* JDK 1.8を使用して実行。
* EXIF方向タグのサポート。
* 一時的なFTPファイルのアップロードディレクトリファイルスキップ（ProFTPで作成されたもの）。
* 画像レンダリングでインスタント公開がサポートされるようになりました。 つまり、レンダリングされたビネットアセットを、公開ジョブを開始する必要なく、Webサイト上で即座に起動できます。 この機能の使用に関心がある場合は、テクニカルサポートに連絡して、すぐに公開するためのアカウントを有効にしてもらってください。

**サポートされなくなった機能**

* IPS Classic公開検証内の「カタログサーバーの再同期」オプションが削除されました。
* 空または重複するメタデータタグフィールドの値を設定できなくなりました。

**バグの修正**

* API `getAssets`に応答して、[!DNL .vtt]ファイルの`<metadataArray>`が空でした。

* 向きのユーザーデータフィールドが見つかりません。
* ACOファイルを更新するジョブに対する`getJobLogDetails`からの応答で、ジョブの詳細が正しく指定されていませんでした。
* PSDプライマリファイルは、Image Serverごとに複数回公開されたと記録されました。
* 名前に無効な文字が含まれるアセットをIPSにアップロードできる。
* 生成に失敗したビデオサムネールが、「キュー内の保留中」の状態のままでした。
* 大きな`deleteFolder`とサーバーディレクトリのアップロードの間のフォルダー復元との競合状態。
* アップロード時にPDFファイルがパブリッシュサーバーごとに2回公開されてログに記録される問題を修正しました。
* 同じレイヤー名を持つ複数のPSDファイルをアップロードすると、最適化エラーが発生していました。
* 顧客PDFのラスタライズで、生成された画像に予期しない白い線が表示されていました。
* ビデオサムネールファイルがアップロード時に複数回公開されていました。
* 重複アップロードファイルの検出で、2つのアップロード通知が生成され、完了したファイルアカウンティングがスローされました。
* ビデオトランスコード用のSidecar XMLファイルが、更新時に複数回公開されたと記録されました。
* ネストされたフォルダーを削除し、（ネストされたフォルダーの親フォルダーになく、ネストされたフォルダーと同じ名前を持つ）別のフォルダーを、ネストされたフォルダーの以前の場所に移動した場合、moveFolderは失敗しました。
* 共有構成およびローカル構成のクラスター状態構成エントリは、クリーンアップを通じてパージされませんでした。
* APIログパーサーを変更して、電子メールアドレス形式をフィルターします。
* ビデオ処理で`Iptc4xmpCore:Location`が正しくマッピングされない。
* クリーンアップとアップロードの間に競合状態が存在していました。
* 公開では`assetFile`が使用されないので、ステージング公開は新しいアセットに対してのみ機能します。
* RuleSetの更新時にNullポインタがスローされました。

## サポートされなくなった機能 {#section-9295ea9313c84695b19bca0e0f01a690}

* **Flashビューアのサポート終了に関するお知らせ2017** *年1月31日に、Adobe Scene7(現在のDynamic Media Classic)はFlashビューアプラットフォームのサポートを正式に終了しました。*

* 注意：2014年1月31日をもって、Scene7はDHTMLビューアプラットフォームのサポートを正式に終了しました。
* Scene7Creative Suite拡張機能は、Creative CloudアプリケーションのFlashパネルのサポート解除により、このリリースで非推奨（廃止予定）となりました。 コンテンツ取り込みワークフローのさらなる開発は、Adobe Experience Manager Dynamic MediaとCreative Cloudに焦点を当てています。 Scene7の場合、Scene7 Publishing Systemのアップロードおよび書き出し機能は引き続き使用できます。
* UGCビデオは、採用が少ないため、このリリースで非推奨（廃止予定）となりました。 別のDynamic Media Classicアカウントを設定することが、UGCビデオの代わりとなるソリューションとなります。 別のAdobeを設定する場合は、アカウントテクニカルサポートにお問い合わせください。
* 採用が少ないので、テキストハイフネーションは、このリリースで非推奨（廃止予定）となりました。 ハイフネーション辞書は、最適なテキストフローをサポートするために、単語を分割およびハイフネーション処理する方法を定義しました。
* メタデータサーバーと類似性検索は、共に採用が少ないため、2015年に廃止されました。 正式な廃止予定日が発表されると、影響を受けるお客様には、Adobeテクニカルサポートからお知らせします。

# Scene7 Publishing System 6.6 リリースノート{#scene-publishing-system-release-notes}

Adobe Experience CloudのAdobe Scene7 Publishing System 6.6 パートの最新の機能、機能強化、および修正について説明した最新のリリースノートです。

## 重要なお知らせ：DHTML ビューアの提供終了 {#section-58966909d9ec4eacbb6153b83b1642d3}

***2014年1月31日より、Scene7はDHTML ビューアプラットフォームのサポートを正式に終了しました。***

## 新しいソフトウェア機能、機能強化、および修正 {#section-30a6201e50b74fc194c08bd732672ac1}

**アセット管理と公開**

* 即時の公開

  公開前にアセットにマークを付ける必要がなくなるため、公開プロセスのスピードと予測可能性が向上します。 徐々に広げられています。

* 大規模なアセットリポジトリを使用している顧客の検索パフォーマンスが向上しました。
* 自動セット生成で、フォルダーの選択が可能になりました。
* 公開された静的アセットのURLを生成する際の操作性が向上しました。

**閲覧者**

* HTML5 SDK

  HTML5 Software Development KitのScene7 Viewerに、次の機能と機能強化が追加されました。

   * スウォッチの処理（最初または最後のスウォッチのドラッグ、スウォッチの整列、スウォッチの1 ページを一度にスクロール、コンテナの塗りつぶしなど）。
   * ズームビュー、ズームターゲット、スウォッチ、ナビゲーションビュー、スピンビューでの透明度のサポートを追加しました。
   * JavaScript（prototype.js）との互換性。
   * Internet ExplorerでのjQueryとの互換性。
   * eCatalog ページスクラバー、グリッドビュー、ページビュー、スウォッチのサムネールに対して行われた機能強化。
   * トラブルシューティング用のサポートとトレース文のログ記録。

* すぐに使用できるHTML5 ビューア

   * デスクトップコンピューターや複数のモバイルデバイスで再生できる機能を備えたビデオプレーヤーを、単一のプラットフォームから開発できるようになりました。 この機能は、Scene7 Viewer、HTML5 Software Development Kit、バージョン 6.5を介して作成できます。
   * 新しい埋め込みフライアウトズーム（FlyoutViewer）画像ビューアが、デスクトップコンピューターとモバイルデバイスで使用できるようになりました。

* 埋め込みコード

   * ユニバーサルビューア埋め込みSDKは、Scene7 パブリッシングシステムと統合されました。
   * JavaScriptを使用したHTML5 ビューアの埋め込み（iframeではない）。
   * ユニバーサルビューア埋め込みのアセットおよびビューア修飾子のサポートが追加されました。
   * SiteCatalystによるトラッキングなど、ユニバーサルビューアの埋め込みコードでJavaScript関数に対するサポートが追加されました。

* S7Info、S7Search、およびS7InfoAdminに様々な機能強化を行うことで、eCatalog機能の信頼性が向上しました。

**アダプティブ Creativeとイメージング**

* アセットの取り込みと準備の効率を向上。

   * 取り込み時に不透明な画像の背景を削除します。

     ピクセルベース： 内部エリアのユーザー定義可能な処理。
   * パス境界に対する自動切り抜きの機能強化。

     ピクセルベース： 左下隅に透明なピクセルを配置する必要はありません。

* Web To Printの機能強化

   * フォントをアウトラインに変換します。
   * FXG変換時にソース画像のフォーマットとカラースペースを維持します。

**セキュリティの強化**

* SPS セキュリティの機能強化

   * SSLのサポート（通信が暗号化されていない領域の場合）
   * アカウント列挙：ユーザー名の送信に対するチャレンジ応答テスト
   * メールサーブレット認証は、メール送信へのアクセスを制限します
   * レコードログインサーブレットへのアクセス制限を削除しました
   * 暗号化されたリンクによるパスワードリセット
   * Segregate DNS （SPSおよびImage Server）

* IPS セキュリティの強化

   * API要求を行っているすべてのオリジナルアプリケーションをSSLに移動するクライアントに焦点を当てたリクエスト

**eVideo**

* 新しいユニバーサル HTML5 Video Playerを使用すると、HTMLでボタンや再生タイマーなど、すべてのビデオコンポーネントを1回設計してから、複数のデバイスやスクリーンで最適なビデオ再生体験を提供できます。

* 1つのビデオファイルを読み取り、複数のエンコーディングに出力するビデオエンコーディングのターンアラウンドタイムを改善しました。

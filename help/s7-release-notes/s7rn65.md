# Scene7 Publishing System 6.5 リリースノート{#scene-publishing-system-release-notes}

Adobe Experience CloudのAdobe Scene7 Publishing System 6.5 パートの最新の機能、機能強化、および修正について説明した最新のリリースノートです。

## 新しいソフトウェア機能、機能強化、および修正 {#section-30a6201e50b74fc194c08bd732672ac1}

**アセットの公開に関する重要な情報**

Scene7の公開プロセスは、2013年第1四半期および第2四半期に変更されます。 「公開と公開のためのマーク」の現在の2段階プロセスが、1段階のプロセスに合理化されました。 「公開用にマーク」ステータスは存在しなくなり、アセットは「公開済み」ステータスまたは「未公開」ステータスになります。 アセットを公開用にマークすると、アセットが自動的に公開されます。

**アセット管理と公開**

* 自動セット生成

  アセットを選択し、自動セット生成を独立したジョブとして実行する機能を追加しました。
* Adobe [!DNL Scene7 Publishing System]へのアップロード時にZIP/TAR ファイルのアンパッケージのサポートを追加しました。

  ZIP ファイルとTAR ファイルをSPSにアップロードする場合は、圧縮ファイルに含まれるディレクトリ構造にアセットを展開するように選択できます。

* フォルダーのドラッグ&amp;ドロップのサポートを追加しました。

  フォルダーを移動するには、**[!UICONTROL ファイル]**/**[!UICONTROL フォルダーを移動]**&#x200B;を選択します。

**閲覧者**

* 3つの新しいHTML5 ビューア（MixedMedia_HTML5、FlyoutZoom_HTML5、Mobile_Spin_HTML5）が追加されました（1軸スピンセットと多軸スピンセットの両方をサポート）。
<!-- 
  [More information](http://help.adobe.com/ja_JP/scene7/using/WS6E593DEA-7D81-4cd6-84B0-85E8BB274176.html#WS1c46793299cf21d77e926d1613177f0a020-8000.html).
  -->
* すべてのHTML5 ビューアがデスクトップでサポートされるようになりました。

<!--   [More information](http://help.adobe.com/ja_JP/scene7/using/WS6E593DEA-7D81-4cd6-84B0-85E8BB274176.html#WS1c46793299cf21d77e926d1613177f0a020-8000.html). -->
* Scene7 Viewer、HTML5 Software Development Kitに次の機能強化が追加されました。

   * JavaScript Libraryの機能強化

     ほとんどのコンポーネントに[!DNL Internet Explorer] 7、8、9のサポートを追加しました。 コンポーネントのJavaScript ファイルに埋め込まれたデフォルトのCSS プロパティ。 同じweb ページで複数のビューアをサポートするようにJSON リクエストを修正しました。

   * ボタンとスウォッチのツールヒントのサポートを追加しました。
   * FlyoutZoomView コンポーネントでのオーバーレイメッセージのサポートを追加しました。
   * SpinViewの機能強化

     2次元スピンセットのサポートを追加しました。 SpinView コンポーネントの画像プリロードを調整しました。 SpinView コンポーネントの画像の入れ替えを調整しました。 1次元および2次元インラインスピンセットのサポートを追加しました。

* Web ページでのビューアのデプロイメントを簡素化するために、**[!UICONTROL 埋め込みコード]**&#x200B;の広範な機能強化が行われました。 次の機能が強化されました。

   * 同じweb ページで複数のユニバーサルビデオビューアをサポートするようになりました。
   * Web ページ上の複数の既存のAS3/AS2 ビューアのサポートを追加しました。
   * DIVの背景色を設定する機能を追加しました。
   * 透明度を適用する機能が追加されました。
   * カスタムアイコンオーバーレイの設定機能を追加しました。
   * 埋め込みビューア内の画像/アセットを入れ替えるためのシンプルなAPI。
   * 非Flash対応デバイスのカスタムフォールバック。
   * サーバー側でのデフォルトの設定属性の保存。
   * ページのDHTML フライアウトビューアーインジェクション。

* ローカライズされたビューアを簡単に作成およびデプロイして、グローバルにリーチできるようになりました。 この機能は、Scene7 Viewer、HTML5 SDKを介して実行されます。

**画像サービング**

* ファイルをアップロードする際に、アンシャープマスクオプションを使用できます。
* シャープニングは、画像の領域内で配信時に適用できます。

**ユーザー生成コンテンツ**

* PDFは、サポートされているファイル形式になりました。

<!--   [More information](http://help.adobe.com/ja_JP/scene7/using/WSe8b0455615e2dc47-2df907a712f31201b35-8000.html).  -->
* ユーザー生成コンテンツのパフォーマンスが5倍以上に向上。

**画像オーサリング**

* 新しいビネット自動化モジュール（s7vampy）は、ビネットの作成を自動化します。

**コンテンツテンプレート（動的バナーとWeb-To-Print）**

* Dynamic Bannersでは、ActionScript/Javascriptのクロスコンパイルなど、HTML 5のサポートが強化されています
* [!DNL Dynamic Banners]のワンクリックプレビューをFirefoxで利用できるようになりました
* **[!UICONTROL Web-To-Print]**&#x200B;では、PDF出力でハイパーリンクを動的に作成できるようになりました
* **[!UICONTROL Web-To-Print]**&#x200B;は、[!DNL OpenType®]個のコンテクスト合字と[!DNL Adobe Illustrator]および[!DNL Adobe InDesign]個のフィルター効果をサポートしています

**eVideo**

* **[!UICONTROL アプリケーション設定]**&#x200B;の&#x200B;**[!UICONTROL 一般設定]** ページの新しい&#x200B;**[!UICONTROL エンコード済みビデオを表示]** オプションでは、ビデオ アセットを単一のアセットタイプに統合して、検索とブラウジングを合理化できるようになりました。

**メタデータと類似性検索**

* 機能&#x200B;**[!UICONTROL 類似性検索]**&#x200B;は、いくつかの技術的な問題により、[!DNL Scene7 Publishing System]に含まれなくなりました。 [!DNL Adobe]は、Scene7への今後の統合のために、イメージング技術への代替投資を行う予定です。

**Adobe Digital Marketing Suiteとの連携**

* [!DNL Scene7]は、パーソナライズ用に[!DNL Adobe Test&Target]件のHTML オファーをサポートするようになりました。 [!DNL Test&Target]個のプロファイルスクリプトとキャンペーンルールに基づいて、関連性の高いバナーコンテンツのオファー要素を動的に組み立てます。

* [!DNL Adobe CQ]との統合の拡張により、フォルダーベースのアセットリポジトリの同期と統一されたコンテンツインターフェイスが提供されます。 [!DNL Scene7] アカウント内の公開された動的アセットにアクセスし、参照して、CQ ページテンプレートに直接挿入します。

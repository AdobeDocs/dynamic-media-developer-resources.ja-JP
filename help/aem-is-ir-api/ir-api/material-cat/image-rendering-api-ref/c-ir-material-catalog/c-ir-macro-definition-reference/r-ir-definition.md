# 定義{#definition}

マクロ定義。 レンダリングサービングプロトコルフラグメントは、「&amp;」文字で区切られた 1 つ以上の完全なコマンドで構成されます。

マクロ定義は、他のマクロを参照できません。 マクロは、カスタム変数を定義および使用できます。

## プロパティ {#section-2584a79bd2a04ddeaa2c04fdac4835ae}

テキスト文字列 必須。

## 初期設定 {#section-4ab127aa653b4f76afa867c1acb1f31c}

なし

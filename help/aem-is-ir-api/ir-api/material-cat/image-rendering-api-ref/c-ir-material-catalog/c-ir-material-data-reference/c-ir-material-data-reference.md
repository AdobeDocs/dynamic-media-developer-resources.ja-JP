# 材料データ参照{#material-data-reference}

マテリアルカタログにマテリアルデータが必要な場合は、マテリアルデータファイルの相対パスまたは絶対パスをattribute::CatalogFileで指定する必要があります。

特定のマテリアルカタログのマテリアルデータファイルのエントリは、既定のカタログのマテリアルデータファイルのエントリを上書きします。

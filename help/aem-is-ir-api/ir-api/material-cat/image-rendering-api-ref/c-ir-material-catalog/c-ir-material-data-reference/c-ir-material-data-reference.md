# 材料データ参照{#material-data-reference}

マテリアルカタログにマテリアルデータが必要な場合は、マテリアルデータファイルの相対パスまたは絶対パスをattribute::CatalogFileに指定する必要があります。

特定の材料カタログの材料データファイル内のエントリは、既定のカタログの材料データファイル内のエントリを上書きします。

# 材料データ参照{#material-data-reference}

材料カタログに材料データが必要な場合、材料データファイルの相対パスまたは絶対パスを attribute::CatalogFile で指定する必要があります。

特定の材料カタログの材料データ ファイル内のエントリは、既定のカタログの材料データ ファイル内のエントリを上書きします。

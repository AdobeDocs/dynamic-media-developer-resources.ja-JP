# BaseColor{#basecolor}

主要な画像カラー： テクスチャの色付けに使用される減法混色。 テクスチャ画像の平均的なドミナントカラーにする必要があります。

また、`catalog::Color`を持たないテクスチャ マテリアルをテクスチャリングできないビネット オブジェクトに適用する場合は、`BaseColor`を塗りつぶしカラーとして使用することもできます。

## プロパティ {#section-d9aec235f28b402581cf2277f5fc6fbb}

カラー： 単色およびキャビネットのマテリアルでは無視されます。

## 初期設定 {#section-ca1813a6fcb24645b5835af0891b279d}

`0x808080` （ニュートラルグレー）。

# RenderSettings{#rendersettings}

高度なレンダリング設定。 高度なレンダリング設定を使用して、レンダリングエンジンの低レベルの側面（マテリアルのシャープニングタイプやパラメーター、イルミネーションの特定のパラメーターなど）を制御します。

## プロパティ {#section-b4c8fe595efc4838ac598659bc820607}

テキスト文字列。 すべてのマテリアルに対してオプションです。 指定する場合は、Vignette オーサリングツール（Dynamic Media画像オーサリングパッケージの一部）で定義されている有効なレンダリング設定文字列である必要があります。

## 初期設定 {#section-6a4d2013c1d34284b4ff21bb07485d28}

`attribute::RenderSettings`が指定されていない場合、または空の場合。

# Repeat{#repeat}

テクスチャ繰り返しモード テクスチャイメージをタイル状にしてターゲットサーフェスを塗りつぶす方法を指定します。

## プロパティ {#section-cef4109cddf54ce095c3293d85bc412d}

列挙。 繰り返し可能なテクスチャにのみ使用します。 その他のマテリアルでは無視されます。

繰り返し可能なテクスチャマテリアルには、次の値を指定できます。

# ID{#id}

カタログレコードのID: マテリアル レコードがレンダーサーバーによって参照されるキー値。 通常は、SKU番号などの短く一意のマテリアル識別子を使用します。

## プロパティ {#section-9b20bf2545314620808b4210bce7afc9}

テキスト文字列。 必須。 各`catalog::Id`値は、このマテリアル カタログ内で一意である必要があり、「,」文字を含めることはできません。

## 初期設定 {#section-6a517046c43a40a7ba0af0212b5f31d2}

なし

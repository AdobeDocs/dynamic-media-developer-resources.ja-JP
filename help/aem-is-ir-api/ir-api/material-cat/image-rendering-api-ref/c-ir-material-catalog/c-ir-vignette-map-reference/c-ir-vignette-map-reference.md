# ビネットマップ参照{#vignette-map-reference}

マテリアルカタログにビネットマップが必要な場合は、ビネットマップファイルの相対パスまたは絶対パスをattribute::VignetMapFileに指定する必要があります。

特定のマテリアルカタログのビネットマップファイル内のエントリは、初期設定のカタログのビネットマップファイル内のエントリを上書きします。

# ビネットマップリファレンス{#vignette-map-reference}

ビネットマップがマテリアルカタログに必要な場合は、ビネットマップファイルの相対パスまたは絶対パスを attribute::VignetMapFile で指定する必要があります。

特定のマテリアルカタログのビネットマップファイルのエントリは、既定のカタログのビネットマップファイルのエントリを上書きします。

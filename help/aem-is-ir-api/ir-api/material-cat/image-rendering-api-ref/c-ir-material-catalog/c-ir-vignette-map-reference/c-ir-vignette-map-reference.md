# ビネットマップリファレンス{#vignette-map-reference}

ビネットマップがマテリアルカタログに必要な場合は、ビネットマップファイルの相対パスまたは絶対パスをattribute::VignetteMapFileで指定する必要があります。

特定のマテリアルカタログのビネットマップファイル内のエントリは、既定のカタログのビネットマップファイル内のエントリを上書きします。

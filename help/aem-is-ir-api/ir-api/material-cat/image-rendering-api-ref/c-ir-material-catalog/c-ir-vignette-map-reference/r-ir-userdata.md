# UserData *{#userdata}

ユーザーデータ： サーバーは、req=userdataに応答して、このフィールドの内容をクライアントに返します。

新しい行は「??」でエスケープできます。 サーバーは、このフィールドの内容を返す前に、&#39;??&#39;を`<cr><lf>`に置き換えます。

## プロパティ {#section-4e3904e047a84cc790ce77c56d3e0b21}

テキスト文字列。 オプション。

## 初期設定 {#section-a84b821918af41f5b84200f2edffacff}

なし

## 関連項目 {#section-f482adf424e44df1b817eb182040fa3a}

# RenderSettings{#rendersettings}

デフォルトの詳細レンダリング設定。 `catalog::RenderSettings`の既定値。

## プロパティ {#section-fa89db09aeeb420292cf783f07b8547a}

テキスト文字列。 指定する場合は、Vignette オーサリングツール（Dynamic Media画像オーサリングパッケージの一部）で定義されている有効なレンダリング設定文字列である必要があります。

## 初期設定 {#section-4e76784aeecd4f09a33124aded7406cb}

`default::RenderSettings`が指定されていない場合、または空の場合。

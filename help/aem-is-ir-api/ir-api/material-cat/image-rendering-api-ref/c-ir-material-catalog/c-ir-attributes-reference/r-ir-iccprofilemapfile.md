# IccProfileMapFile{#iccprofilemapfile}

ICC プロファイルマップファイルパス。 この画像カタログのICC プロファイルマップを含むファイルを指定します。

## プロパティ {#section-33d55b55e192441bb30ca5eaaf07d12a}

テキスト文字列値。 カタログフォルダーに対する絶対ファイルパスまたはパス。

## 初期設定 {#section-191de4cf304544f0a4198ebd3b8ddfc2}

空。このマテリアル カタログにプライベート ICC プロファイルが含まれていないことを示します。

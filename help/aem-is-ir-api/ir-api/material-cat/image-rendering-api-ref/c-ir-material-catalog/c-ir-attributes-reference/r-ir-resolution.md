# 解像度{#resolution}

デフォルトの解像度： 特定のカタログレコードに有効な`catalog::Resolution`値が含まれていない場合に備えて、デフォルトの解決策を提供します。

## プロパティ {#section-06d519158b9f479896f945747c670736}

実数（`0`より大）。 通常、1 インチあたりのピクセル数で表されますが、1 メートルあたりのピクセル数など、他の単位でも表すことができます。

## 初期設定 {#section-eea922c37c224e1dbcab3bc53ee13aca}

定義されていない場合や空の場合は、`default::Resolution`から継承されます。

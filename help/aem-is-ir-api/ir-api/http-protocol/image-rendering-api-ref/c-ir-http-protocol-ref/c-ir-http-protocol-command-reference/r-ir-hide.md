# 非表示{#hide}

重なりオブジェクトを非表示にします。 現在の選択範囲のウィンドウのカバー枠に適用されるすべてのオーバーラップオブジェクトまたはウィンドウのカバー範囲を非表示にします。

`hide[= *`val`*]`

 <tr class="strow"> 
  <td class="stentry"> <p><span class="codeph"> <span class="varname"> val</span></span> </p> </td> 
  <td class="stentry"> <p>無視。 </p></td> 
 </tr> 
</table>

## プロパティ {#section-2952fa43df0547b187b834c819c517ca}

現在のグループまたはオブジェクトの選択範囲に適用されます。 現在の選択範囲に重なり合うオブジェクトまたはウィンドウのカバー枠が含まれていない場合、または重なり合うオブジェクトが表示されない場合は無視されます。

## 初期設定 {#section-9da202944d3a49b7ae81c78370b2b5e7}

指定しない場合、選択した重なりオブジェクトの現在の表示が保持されます。

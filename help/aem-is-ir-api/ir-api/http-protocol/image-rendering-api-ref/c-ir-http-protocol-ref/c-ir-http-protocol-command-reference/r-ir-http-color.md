# color {#color}

描画色： 単色マテリアルの色または着色可能なマテリアルの加法混色を指定します。

`color= *`color`*`

<table id="simpletable_C5AF9074CCA64EA5921772DF3F7E0F55"> 
 <tr class="strow"> 
  <td class="stentry"> <p><span class="varname"> カラー</span> </p> </td> 
  <td class="stentry"> <p>RGBまたはグレーの値。 </p></td> 
 </tr> 
</table>

## プロパティ {#section-629c3c91221c48c4b7f7b31a13fd1766}

マテリアル属性： 単色の素材に必要です。他のすべての素材に対してオプションです。

## 初期設定 {#section-ea8e1967674d426bb8f46abe365b6aca}

`catalog::Color` マテリアルがカタログ エントリに基づいている場合。 それ以外の場合は、色付けなしで`none`。

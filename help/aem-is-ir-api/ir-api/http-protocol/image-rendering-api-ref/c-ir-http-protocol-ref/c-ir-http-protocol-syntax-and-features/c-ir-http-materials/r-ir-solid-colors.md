# べた塗{#solid-colors}

このマテリアルカテゴリには、壁、木工、トリム、自動車、その他のテクスチャが必要ないアプリケーション用のペイントが含まれます。

例えば、テクスチャの変更が不要なアパレルのアイテムの色を変更する場合や、自動車のインテリアのプラスチックトリムやカーペットの色を設定する場合に、単色を使用します。 MSSに`src=`値が存在しない場合、マテリアルは単色と見なされます。

べた塗りのカラーは、すべてのレンダリング可能なオブジェクトに適用できます。

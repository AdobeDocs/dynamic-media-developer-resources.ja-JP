# 選択コマンド{#selection-commands}

これらのコマンドを使用して、周辺光量補正グループ、オブジェクト、グループまたはオブジェクトのサブエリアを選択します。

この選択コマンドの後、次の選択コマンド（またはリクエストの最後）が選択された項目に適用される前に、コマンド、またはマテリアル、またはその両方が指定されます。 選択コマンドは、マテリアル仕様セグメント（MSS）を区切ります。

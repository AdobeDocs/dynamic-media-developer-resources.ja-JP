# 材料の解像度{#material-resolution}

繰り返し可能なテクスチャや他のマテリアルの解像度の値によって、シーンの座標空間でのマテリアルイメージのサイズが定義されます。

テクスチャ解像度の値は、1 インチあたりのピクセル数（ビネットがシーンの座標空間に指定する単位）で指定します。

材料の解像度は、 `res=`.

# 材料解像度{#material-resolution}

繰り返し可能なテクスチャや他のマテリアルの解像度の値によって、シーンの座標空間でのマテリアルイメージのサイズが決まります。

テクスチャ解像度の値は、ピクセル/インチ（またはビネットがシーン座標空間に対して指示する単位）で指定します。

材料の解像度は`res=`で指定します。

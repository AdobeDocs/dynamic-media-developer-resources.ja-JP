# 材料の解像度{#material-resolution}

繰り返し可能なテクスチャまたはその他のマテリアルの解像度の値によって、シーン座標空間のマテリアル イメージのサイズが定義されます。

テクスチャ解像度の値は、1 インチあたりのピクセル数（またはビネットがシーン座標空間に対して指示する単位）で指定します。

材料分解能は `res=` で指定します。

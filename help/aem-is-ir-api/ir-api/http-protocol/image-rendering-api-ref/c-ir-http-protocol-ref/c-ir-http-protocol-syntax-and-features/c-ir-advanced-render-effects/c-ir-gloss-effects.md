# 光沢効果{#gloss-effects}

ビネットで有効にした場合、レンダラーは照明アルゴリズムを変更して、（主にコントラストを制御することで）マテリアル光沢の変化をシミュレートします。

この効果は反射レンダリングとは独立し、鏡面反射をシミュレートして補完します。 効果は、反射を制御するのと同じ`gloss=`、`glossmap=`および`type=`属性を使用して制御します。

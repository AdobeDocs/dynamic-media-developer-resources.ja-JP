# 光沢効果{#gloss-effects}

ビネット内で有効になっている場合、レンダラーは照明アルゴリズムを変更して、マテリアルの光沢の変化をシミュレートします（主にコントラストを制御することによって）。

この効果は反射レンダリングとは無関係で、鏡面反射をシミュレートすることによって効果を補完します。 エフェクトは、反射の制御に使用するのと同じ `gloss=`、`glossmap=`、`type=` のアトリビュートで制御されます。

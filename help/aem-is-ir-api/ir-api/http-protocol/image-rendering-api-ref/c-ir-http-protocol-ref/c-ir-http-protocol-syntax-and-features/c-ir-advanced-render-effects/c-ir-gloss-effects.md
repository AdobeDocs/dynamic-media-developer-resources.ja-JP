# 光沢効果{#gloss-effects}

ビネットで有効にした場合、レンダラは照明アルゴリズムを変化させて、マテリアルの光沢の変化（主にコントラストの制御）をシミュレートします。

この効果は反射レンダリングとは無関係で、鏡面反射をシミュレートすることで補完します。 効果は同じで制御される `gloss=`, `glossmap=`、および `type=` 反射の制御に使用する属性。

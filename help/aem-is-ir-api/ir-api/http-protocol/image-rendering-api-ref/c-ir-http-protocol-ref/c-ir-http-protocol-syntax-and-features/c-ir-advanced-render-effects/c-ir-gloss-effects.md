# 光沢効果{#gloss-effects}

周辺光量補正で有効になっている場合、レンダラーは照明アルゴリズムを変更して、マテリアルの光沢度のバリエーションをシミュレートします（主にコントラストを制御します）。

このエフェクトは、反射レンダリングとは独立しており、スペキュラ反射をシミュレートすることで補完します。 効果は、反射の制御に使用されるのと同じ`gloss=`、`glossmap=`、`type=`属性で制御されます。

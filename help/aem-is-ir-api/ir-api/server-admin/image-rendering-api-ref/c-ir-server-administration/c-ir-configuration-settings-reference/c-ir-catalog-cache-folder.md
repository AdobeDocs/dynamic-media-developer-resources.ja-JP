# カタログキャッシュフォルダ(ir.catalogCacheRoot){#catalog-cache-folder-ir-catalogcacheroot}

絶対パスまたは[!DNL *[!DNL install_folder]*]に対する相対パスを指定できます。 既定値 [!DNL ./cache]は、[!DNL *[!DNL install_folder]*/cache]に対応します。

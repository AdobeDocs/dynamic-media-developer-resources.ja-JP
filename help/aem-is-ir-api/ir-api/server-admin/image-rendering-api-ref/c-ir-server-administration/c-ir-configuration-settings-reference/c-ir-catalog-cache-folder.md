# カタログキャッシュフォルダー(ir.catalogCacheRoot){#catalog-cache-folder-ir-catalogcacheroot}

絶対パスまたは[!DNL *[!DNL install_folder]*]に対する相対パスを指定できます。 初期設定は[!DNL ./cache]で、[!DNL *[!DNL install_folder]*/cache]に対応します。

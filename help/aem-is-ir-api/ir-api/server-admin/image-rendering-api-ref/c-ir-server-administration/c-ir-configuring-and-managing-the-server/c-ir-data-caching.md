# データキャッシュ{#data-caching}

画像レンダリングでは、画像サービングのキャッシュ機能を利用して、応答画像データ、ネスト/埋め込み要求の中間データ、外部ソースから取得したソース画像データ（テクスチャなど）を保存します。

キャッシュメカニズムについて詳しくは、画像サービングのドキュメントを参照してください。

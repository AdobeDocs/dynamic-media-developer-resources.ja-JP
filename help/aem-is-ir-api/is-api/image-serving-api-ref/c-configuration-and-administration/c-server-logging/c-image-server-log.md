# Image Server ログ{#image-server-log}

Image Serverはこのログファイルにレコードを書き込み、警告とエラーを記録し、起動イベントとシャットダウン イベントの記録を保持します。

さらに、Image Server ログを設定して、画像ファイルへのアクセスなどの実質的な追加の詳細を追跡し、サーバーの問題のデバッグに役立てることができます。

Image Server ログは[!DNL ImageServerRegistry.xml]に設定されています。

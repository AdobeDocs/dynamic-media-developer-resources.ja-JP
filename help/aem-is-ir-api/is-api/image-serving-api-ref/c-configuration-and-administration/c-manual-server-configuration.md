# Server configuration{#server-configuration}

画像サービングと画像レンダリングの多くの要素は、適切な設定ファイルを編集することで、手動で設定できます。

詳しくは、画像サービングと画像レンダリングのドキュメントの管理の節を参照してください。 [!DNL Jconsole] は、ポート9999でも使用できます。

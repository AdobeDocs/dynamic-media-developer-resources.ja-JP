# サーバーの設定{#server-configuration}

適切な設定ファイルを編集することで、画像サービングおよび画像レンダリングの多くの側面を手動で設定できます。

詳しくは、画像サービングと画像レンダリングのドキュメントの管理の節を参照してください。 [!DNL Jconsole] は、ポート 9999 でも使用できます。

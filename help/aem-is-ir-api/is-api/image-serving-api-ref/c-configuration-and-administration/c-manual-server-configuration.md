# サーバ構成{#server-configuration}

画像サービングと画像レンダリングの様々な要素は、適切な設定ファイルを編集することで手動で設定できます。

詳しくは、画像サービングと画像レンダリングのドキュメントの管理の節を参照してください。 [!DNL Jconsole] は、ポート9999で使用することもできます。

# imageRes{#imageres}

Web To Print 要求に埋め込まれたビットマップの解像度を設定します。

` imageRes=< *` 数値 `*>`

印刷するときは 300 をお勧めします。 数値が大きいほど、イメージのレンダリングに必要な時間が長くなります。

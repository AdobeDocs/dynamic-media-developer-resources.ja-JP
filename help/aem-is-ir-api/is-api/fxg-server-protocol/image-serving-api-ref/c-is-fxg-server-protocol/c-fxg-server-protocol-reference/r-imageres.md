# imageRes{#imageres}

Web印刷用リクエストに埋め込まれたビットマップの解像度を設定します。

` imageRes=< *`番号`*>`

印刷する場合は300をお勧めします。 数が多いほど、イメージのレンダリングに必要な時間が長くなります。

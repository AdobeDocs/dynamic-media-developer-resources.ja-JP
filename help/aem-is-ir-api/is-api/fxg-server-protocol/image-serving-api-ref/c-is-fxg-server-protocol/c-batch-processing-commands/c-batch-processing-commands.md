# バッチ処理コマンド{#batch-processing-commands}

これらは、特別なリクエスト形式で`?req=<command>`として指定されます

これらのコマンドの例では、架空の`PlatformServer` URL 「[!DNL scene7.adobe.com:8080]」とAcmeCorpという架空の会社を使用しています。 ジョブ IDも架空のものですが。

* [batchjobdelete](r-batchjobdelete.md)
* [batchjobdetailedstatus](r-batchjobdetailedstatus.md)
* [batchjobgetoutput](r-batchjobgetoutput.md)
* [batchjobsubmit](r-batchjobsubmit.md)

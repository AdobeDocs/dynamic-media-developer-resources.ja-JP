# jobid{#jobid}

バッチジョブのジョブ ID。

バッチジョブリクエストで、ステータスまたは出力がリクエストされるジョブを指定したり、特定のジョブを削除したりするために使用されます。 値は、ジョブの送信時に返されたジョブ ID 文字列です。

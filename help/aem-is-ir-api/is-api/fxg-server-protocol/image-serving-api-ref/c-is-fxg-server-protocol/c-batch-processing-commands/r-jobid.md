# jobid{#jobid}

バッチジョブのジョブID。

バッチジョブリクエストで使用され、ステータスまたは出力をリクエストするジョブを指定するか、特定のジョブを削除します。 値は、ジョブが送信されたときに返されたジョブID文字列です。

# 斜体{#italic}

イタリア国旗。

RTF斜体(`\i`)コマンドによって要求された場合に、このフォントマップレコードで参照されるフォントファイルを使用するかどうかを指定します。

## プロパティ {#section-607e76490e87418496d1c6763018a14f}

フラグ。 フォントマップ用の3次インデックスキー（オプション）。

## 初期設定 {#section-0b88057d00ce4160929c23fbf138b309}

`0.`

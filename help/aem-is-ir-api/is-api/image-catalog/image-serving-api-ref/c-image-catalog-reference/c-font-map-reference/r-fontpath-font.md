# FontPath{#fontpath}

フォントファイルのパス プリンシパルフォントファイルのパスと名前（ファイルサフィックスを含む）。

## プロパティ {#section-ae2b7daf74a4419ebcec8f13aa8c1462}

テキスト文字列。 必須。 有効なImage Server ファイル パスが必要です。絶対パスまたは`attribute::RootPath`からの相対パスを指定してください。

## 初期設定 {#section-3fb42f362de84fdb8ba9fe4cc9628aa3}

なし

# PsName{#psname}

Postscript フォントの名前。

現在のところ、FXG レンダラーのみがPostscript名でフォントを参照できます。

## プロパティ {#section-a560f8bfc69a4d88aeb0b53c1625153b}

テキスト文字列。 オプション。 フォントマップのセカンダリインデックスキー。 同じ`font::PsName`値は、同じフォントマップファイル内で1回だけ発生する可能性があります。

## 初期設定 {#section-c5903d5340e740b69ec0150d2303a893}

なし

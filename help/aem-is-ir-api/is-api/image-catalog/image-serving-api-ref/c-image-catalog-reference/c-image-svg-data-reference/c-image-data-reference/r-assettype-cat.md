# AssetType{#assettype}

アセットタイプ。 catalog::ImageSet に公開されたセットのタイプを識別するために使用します。

アセットタイプは、リクエストに対して生成する応答のタイプ `req=set` 決定します。 値を指定しない場合、自動検出ルールによって応答タイプ `req=set` 決定されます。

## プロパティ {#properties}

次のセットの列挙値：

MEDIASET

スピンセット

SPINSET2D

VIDEOSET

画像セット

ECATALOG

## 初期設定 {#section-b94ce61423194729918456103ef3f72c}

なし

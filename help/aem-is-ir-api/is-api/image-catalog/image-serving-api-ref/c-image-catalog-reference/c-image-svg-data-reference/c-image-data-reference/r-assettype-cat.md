# AssetType{#assettype}

アセットタイプ： カタログ：:ImageSetに公開されたセットのタイプを識別するために使用します。

アセットタイプは、`req=set`要求に対して生成する応答のタイプを決定します。 値が指定されていない場合、自動検出ルールによって`req=set`応答タイプが決定されます。

## プロパティ {#properties}

次のセットの列挙値：

MEDIASET

SPINSET

SPINSET2D

VIDEOSET

IMAGESET

ECATALOG

## 初期設定 {#section-b94ce61423194729918456103ef3f72c}

なし

# Image/SVG データリファレンス{#image-svg-data-reference}

画像データとSVG データは非常に似ていますが、画像カタログ内の別々のテーブルに保持されます。

画像データが必要な場合は、画像を列挙する画像データファイルの相対パスまたは絶対パスを`attribute::CatalogFile`で指定する必要があります。 SVG データが必要な場合は、SVG データ ファイルの相対パスまたは絶対パスを`attribute::SvgCatalogFile`で指定する必要があります。

* [画像データ参照](c-image-data-reference/c-image-data-reference.md)
* [SVG データリファレンス](c-svg-data-reference/c-svg-data-reference.md)

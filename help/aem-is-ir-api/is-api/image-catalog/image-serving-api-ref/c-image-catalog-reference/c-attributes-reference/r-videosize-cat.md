# VideoSize{#videosize}

ビデオサイズ 各ビデオレコードに関連付けられたサイズを指定できます。

`req=set`リクエストによってビデオサイズデータの書き出しに使用されます。

## プロパティ {#section-93a2631a57e2423ea6b40f68c55062a1}

幅と高さのコンマ区切り整数値（例：512,288）。

## 初期設定 {#section-baaee759ef344c54bbc014dc8944768a}

なし

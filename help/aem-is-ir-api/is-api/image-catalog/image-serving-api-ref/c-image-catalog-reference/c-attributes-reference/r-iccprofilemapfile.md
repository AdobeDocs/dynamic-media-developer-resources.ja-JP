# IccProfileMapFile{#iccprofilemapfile}

ICC プロファイルマップファイルパス。 この画像カタログのICC プロファイルマップを含むファイルを指定します。

## プロパティ {#section-962934fded9746c7953beb62616dd2a7}

テキスト文字列値。 カタログフォルダーに対する絶対ファイルパスまたはパス。

## 初期設定 {#section-c368cbef2a24435ea97cc7c83bbe2ce8}

空。この画像カタログにプライベート ICC プロファイルが含まれていないことを示します。

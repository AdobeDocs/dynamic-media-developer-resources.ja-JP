# HttpAppleStreamingContext{#httpapplestreamingcontext}

## プロパティ {#properties}

URLプレフィックスを表す文字列。 (例：`http://my.server.com/apple/streaming`)

## 初期設定 {#section-bdd70081d50844c295f0de0154c1d738}

# ApplyEffectMask{#applyeffectmask}

必要に応じて、エフェクトレイヤーに最終マスクを適用し、エフェクトとレイヤーの間の重なり領域をマスクアウトします。

有効な値は0と1です。 0の場合、効果マスクは適用されません。 1はエフェクトマスクを適用することを意味します。 初期設定は1です。

# ApplyEffectMask{#applyeffectmask}

必要に応じて、エフェクトレイヤーに最終的なマスクを適用し、エフェクトとレイヤーの重なり領域をマスクアウトします。

有効な値は0と1です。 0は、エフェクトマスクを適用しないことを意味します。 1は、エフェクトマスクを適用することを意味します。 初期設定は1です。

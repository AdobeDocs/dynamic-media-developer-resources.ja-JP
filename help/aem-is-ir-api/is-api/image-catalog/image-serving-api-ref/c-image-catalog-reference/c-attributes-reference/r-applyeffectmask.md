# ApplyEffectMask{#applyeffectmask}

必要に応じて、エフェクトレイヤーに最終マスクを適用して、エフェクトとレイヤーの重なり合う領域をマスクします。

有効な値は0と1です。 0は、エフェクトマスクを適用しないという意味です。 1は、エフェクトマスクを適用することを意味します。 デフォルトは1です。

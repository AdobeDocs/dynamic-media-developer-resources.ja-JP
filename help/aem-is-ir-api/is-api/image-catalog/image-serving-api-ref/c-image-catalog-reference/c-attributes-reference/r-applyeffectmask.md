# ApplyEffectMask{#applyeffectmask}

エフェクトレイヤーに最終マスクを適用して、エフェクトとレイヤーの重複領域をマスクすることもできます。

有効な値は 0 と 1 です。 0 は、エフェクトマスクを適用しないことを意味します。 1 はエフェクト マスクを適用することを意味します。 デフォルトは 1 です。

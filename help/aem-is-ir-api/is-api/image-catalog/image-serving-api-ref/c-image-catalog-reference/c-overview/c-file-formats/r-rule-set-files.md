# ルールセットファイル{#rule-set-files}

ルールセットファイルは、XML形式のテキストファイルであり、適切な標準と規則に従う必要があります。

[!DNL RuleSet.xsd]は既定のカタログ フォルダーにインストールされており、ルール セット ファイルを画像サービングに送信する前に検証するために使用する必要があります。 厳密な解析が適用され、[!DNL RuleSet.xsd]に準拠しないルールセットファイルは読み込まれません。

# プロパティ{#properties}

プロパティデータは、次の`req=`種類に応じて返されます。

* `catalogprops`
* `imageprops`
* `props`
* `userdata`

`userdata`件の応答は、`catalog::UserData`の内容がプロパティ形式に従う場合にのみ、プロパティとして書式設定されます。

* [テキスト（Java）プロパティ](r-text-java-properties.md)
* [JavaScript properties](r-javascript-properties.md)
* [XML プロパティ](r-xml-properties.md)
* [JSONP プロパティ](r-json-properties.md)

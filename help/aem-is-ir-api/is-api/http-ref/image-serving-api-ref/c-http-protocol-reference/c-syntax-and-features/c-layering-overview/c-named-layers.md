# 名前の付いたレイヤー{#named-layers}

layer=コマンドを使用すると、レイヤーに名前を割り当て、後でレイヤー番号ではなく名前で同じレイヤーを参照できます。

これにより、テンプレートで名前の付いたレイヤーを定義し、変数を使用しなくても、リクエストURLから参照できるようになります。

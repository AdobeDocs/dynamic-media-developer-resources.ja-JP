# 名前の付いた画層{#named-layers}

layer= コマンドを使用すると、画層に名前を割り当て、後で画層番号ではなく名前で同じ画層を参照できます。

これにより、テンプレートでは名前付きレイヤーを定義できます。このレイヤーは、変数を使用せずに、リクエスト URL から参照できます。

# 名前の付いた画層{#named-layers}

layer=コマンドを使用すると、画層に名前を割り当て、画層番号ではなく同じ画層を名前で参照できます。

これにより、テンプレートで名前の付いたレイヤーを定義し、変数を使用する必要なく、要求URLから参照できます。

# 名前の付いた画層{#named-layers}

layer=コマンドを使用すると、画層に名前を割り当て、後で同じ画層を画層番号ではなく名前で参照できます。

これにより、テンプレートで名前付きのレイヤーを定義し、変数を使用する必要なく要求URLから参照できるようになります。

# レイヤーの順序{#layer-order}

レイヤー番号によって、z順序も決まります。

レイヤー0（背景レイヤー）は必須です。他のレイヤー番号は連続している必要はなく、レイヤー番号を昇順にして背景レイヤーの上に描画されます。 レイヤ番号が最も大きいレイヤは上にレンダリングされ、他のレイヤに隠れることはありません。

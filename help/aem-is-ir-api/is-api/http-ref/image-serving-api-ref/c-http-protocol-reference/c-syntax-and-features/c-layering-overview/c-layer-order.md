# レイヤーの順序{#layer-order}

レイヤー番号によってZ順序も決まります。

レイヤー0 （背景レイヤー）が必要です。その他のレイヤー番号は連続する必要はなく、レイヤー番号の昇順で背景レイヤーの上に描画されます。 最も高いレイヤー番号を持つレイヤーは上にレンダリングされ、他のレイヤーによって隠されることはありません。

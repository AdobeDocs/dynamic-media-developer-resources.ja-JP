# レイヤーの順序{#layer-order}

レイヤ番号によってz順序も決まります。

レイヤー0（背景レイヤー）が必要です。その他のレイヤ番号は、連続する必要はなく、背景レイヤの上に、昇順のレイヤ番号の順に描画されます。 最も画層番号が大きい画層は上にレンダリングされ、他の画層に隠れることはありません。

# レイヤーの順序{#layer-order}

また、レイヤ番号によって Z オーダーも決まります。

レイヤー 0 （背景レイヤー）が必要です。他のレイヤー番号は連続している必要はなく、レイヤー番号の昇順に背景レイヤーの上に描画されます。 レイヤー番号が最も大きいレイヤーが上にレンダリングされ、他のレイヤーによって隠されることはありません。

# レイヤーの順序{#layer-order}

また、レイヤ番号によって z の順序も決まります。

レイヤー 0（背景レイヤー）が必要です。その他のレイヤ番号は、連続する必要はなく、レイヤ番号が昇順の順に背景レイヤの上に描画されます。 最も画層番号が大きい画層は上にレンダリングされ、他の画層に隠れることはありません。

## プロパティ {#section-114ca0b8585b403c873e2251478ad1d5}

テキストレイヤー属性。 画像、べた塗りおよびエフェクトレイヤーでは無視されます。 `layer=comp`に対して指定した場合、`layer=0`に適用されます。

## 初期設定 {#section-855230f5330b4afc8a933f00a1ed4612}

`textAttr=72,norm,fixedRes,wrap`

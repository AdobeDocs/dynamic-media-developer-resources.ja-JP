# bfc{#bfc}

ブラウザーフォーマットの変換 ブラウザー形式変換機能をオンにするかオフにするかを指定します。

`bfc=on|off`

<table id="simpletable_2D23B1B282CD4216AB5BE7E7430D1B3F"> 
 <tr class="strow"> 
  <td class="stentry"> <p> <span class="codeph"> on|off </span> </p> </td> 
  <td class="stentry"> <p>ブラウザー形式変換機能 — JPEGまたはPNG画像を小さいブラウザー固有のファイル形式に変換する — をtrue（オン）またはfalse（ブラウザー固有の形式変換を行わない場合はオフ）にする <span class="codeph"> かどうかを指 </span><span class="codeph"></span> 定します。 </p> </td> 
 </tr> 
</table>

## プロパティ {#section-19dfc13e997f4a33889a1df1e4ad50b9}

リクエスト属性。 JPEGまたはPNGファイルを要求しない場合は無効です。

## 初期設定 {#section-ecc2385ed8c0408cbab3f5cf9c079dad}

`bfc=off`

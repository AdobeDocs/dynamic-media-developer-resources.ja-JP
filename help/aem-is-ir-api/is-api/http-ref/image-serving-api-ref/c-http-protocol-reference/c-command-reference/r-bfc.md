# bfc{#bfc}

ブラウザー形式の変換： ブラウザー形式の変換機能をオンまたはオフにするかどうかを指定します。

`bfc=on|off`

<table id="simpletable_2D23B1B282CD4216AB5BE7E7430D1B3F"> 
 <tr class="strow"> 
  <td class="stentry"> <p> <span class="codeph"> on|off </span> </p> </td> 
  <td class="stentry"> <p>JPEGまたはPNG画像をより小さなブラウザー固有のファイル形式に変換するブラウザー形式の変換機能が、true （</span>の<span class="codeph">）であるか、false （</span>の<span class="codeph">）であるかを指定します。ブラウザー固有のファイル形式を変換しない場合です。 </p> </td> 
 </tr> 
</table>

## プロパティ {#section-19dfc13e997f4a33889a1df1e4ad50b9}

リクエスト属性： JPEGまたはPNG ファイルをリクエストしない場合は効果がありません。

## 初期設定 {#section-ecc2385ed8c0408cbab3f5cf9c079dad}

`bfc=off`

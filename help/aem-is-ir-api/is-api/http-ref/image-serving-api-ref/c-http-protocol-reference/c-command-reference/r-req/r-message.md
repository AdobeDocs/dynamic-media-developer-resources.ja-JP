# message{#message}

クライアントメッセージ。 クライアントが短いテキストメッセージをサーバーログに挿入するメカニズムを提供します。

`req=message&message= *` 文字列 `*`

 </tr> 
</table>

HTTP 応答はキャッシュできません。 空の応答が MIME タイプ `text/plain` と共に返されます。

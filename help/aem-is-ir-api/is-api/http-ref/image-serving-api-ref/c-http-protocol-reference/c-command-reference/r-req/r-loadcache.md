# loadcache{#loadcache}

サーバーキャッシュをプリロードします。 req=imgと同様に要求を実行しますが、画像を返す代わりに、サーバーはMIMEタイプがtext/plainのテキストデータとして形式設定された返信画像(image.length)の長さを返します。

`req=loadcache`

HTTP応答はキャッシュできません。

リクエスト内の他のコマンドは、ドキュメントに従って適用されます。

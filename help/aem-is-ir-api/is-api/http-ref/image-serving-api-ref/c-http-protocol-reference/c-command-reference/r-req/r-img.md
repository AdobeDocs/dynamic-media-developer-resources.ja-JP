# img{#img}

画像（デフォルト）。 標準画像データを要求します。

`req=img`

返信データの形式と応答のMIME タイプは、`fmt=`によって決まります。 修飾子`req=img`は既定の要求タイプであり、明示的に必要ではありません。 HTTP応答は、`catalog::Expiration`に基づくTTLでキャッシュできます。

その他のリクエストコマンドは、文書化されたとおりに適用されます。

# img{#img}

画像（デフォルト）。 標準の画像データをリクエストします。

`req=img`

返信データの形式と応答の MIME タイプは、 `fmt=`. 修飾子 `req=img` はデフォルトのリクエストタイプで、明示的には必要ありません。 HTTP 応答は、TTL に基づいてキャッシュ可能です。 `catalog::Expiration`.

その他のリクエストコマンドは、ドキュメントに従って適用されます。

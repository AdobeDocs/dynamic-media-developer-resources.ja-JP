# img{#img}

Image（初期設定）。 標準の画像データを要求します。

`req=img`

応答データの形式と応答のMIMEタイプは`fmt=`によって決定されます。 `req=img` はデフォルトのリクエストタイプで、明示的に指定する必要はありません。HTTP応答は、`catalog::Expiration`に基づいてTTLでキャッシュ可能です。

その他の要求コマンドは、ドキュメントに従って適用されます。

# effectMask{#effectmask}

リクエストの attribute::ApplyEffectMask の値をオーバーライドします。

effectMask はグローバル モディファイヤです。 リクエスト内のすべてのエフェクトレイヤーに適用されます。

有効な値は 0 と 1 です。 ゼロ（0）は、エフェクトマスクを適用しないことを意味します。 1 は、エフェクトマスクを適用する方法です。 初期設定は 1。

# effectMask{#effectmask}

リクエストのattribute::ApplyEffectMaskの値を上書きします。

effectMaskはグローバル修飾子です。 リクエスト内のすべてのエフェクトレイヤーに適用されます。

有効な値は0と1です。 ゼロ（0）は、エフェクトマスクを適用しないことを意味します。 1つ（1）は、エフェクトマスクを適用することを意味します。 デフォルトは1です。

# op_invert{#op-invert}

色を反転します。 各カラーコンポーネントを、画像の効果が反対になるように反転します。

`op_invert=0|1`

## プロパティ {#section-2e7a7b340dd147db9bd68ff3bfb6f4c1}

[ 画層 ] コマンド 現在の画層または合成画像に適用されます ( `layer=comp`. 効果画層で無視されます。

## 初期設定 {#section-8040bc4e2a684c629a28b155aa0ba289}

`op_invert=0`（カラー反転なし）。

# 非表示{#hide}

レイヤーを非表示にします。 リクエスト文字列からレイヤーを削除せずにレイヤーを非表示にできます。

`hide=0|1`

## プロパティ {#section-09ff67290f064411baeb503a1aac2953}

レイヤー属性。 `layer=comp`に指定すると、レイヤー0に適用されます。

## 初期設定 {#section-ec9f5130be1b435c84ba1bba54e38a48}

表示レイヤーの`hide=0`。

# 非表示{#hide}

レイヤーを非表示 要求文字列からレイヤーを削除せずに、レイヤーを非表示にできます。

`hide=0|1`

## プロパティ {#section-09ff67290f064411baeb503a1aac2953}

レイヤー属性。 に対して指定した場合に、レイヤ0に適用されま `layer=comp`す。

## 初期設定 {#section-ec9f5130be1b435c84ba1bba54e38a48}

`hide=0` を表示します。

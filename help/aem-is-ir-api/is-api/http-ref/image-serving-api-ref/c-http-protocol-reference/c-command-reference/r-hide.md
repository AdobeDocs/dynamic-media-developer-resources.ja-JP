# 非表示{#hide}

レイヤを非表示にする： 要求文字列からレイヤーを削除せずに非表示にできます。

`hide=0|1`

## プロパティ {#section-09ff67290f064411baeb503a1aac2953}

レイヤー属性。 次の項目に指定された場合に、レイヤ 0 に適用されます。 `layer=comp`.

## 初期設定 {#section-ec9f5130be1b435c84ba1bba54e38a48}

`hide=0` 表示される画層の場合。

# 非表示{#hide}

レイヤーを非表示 要求文字列からレイヤーを削除せずに非表示にできます。

`hide=0|1`

## プロパティ {#section-09ff67290f064411baeb503a1aac2953}

レイヤー属性。 `layer=comp`に対して指定された場合、レイヤ0に適用されます。

## 初期設定 {#section-ec9f5130be1b435c84ba1bba54e38a48}

`hide=0` （表示されているレイヤの場合）

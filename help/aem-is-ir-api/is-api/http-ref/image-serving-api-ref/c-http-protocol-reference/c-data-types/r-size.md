 </tr> 
 <tr class="strow"> 
  <td class="stentry"> <p><span class="codeph"> <span class="varname"> px、py</span></span> </p></td> 
  <td class="stentry"> <p>x, y値（ピクセル単位、整数） </p></td> 
 </tr> 
</table>

との両方 *`px`* は0よ *`py`* り大きい値にする必要があります。 `0,0` には、特定のデフォルト値を使用する必要があることを示す場合があります。

# コード{#coord}

ピクセル座標： 画像またはレイヤーの長方形の左上隅を基準としたピクセルオフセットの形式で画像座標を指定するために使用します。 これらの座標は、画像のオフセットや切り抜きパラメーターで頻繁に使用されます。

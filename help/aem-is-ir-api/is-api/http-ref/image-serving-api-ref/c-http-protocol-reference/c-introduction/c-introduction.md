このドキュメントでは、Scene7画像サービングのHTTPプロトコルについて説明します。

プロトコルの公開された側面のみが説明されます。 サーバは、追加のコマンドをサポートする場合があります。これは、Scene7クライアントソフトウェアで使用するために予約されています。

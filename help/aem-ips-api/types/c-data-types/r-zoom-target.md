## パラメーター {#section-2fc9bea56b6d4b72b80d4f04c5f9b862}

| 名前 | 種類 | 説明 |
|---|---|---|
| zoomTargetHandle | `xsd:string` | ターゲットハンドルをズームします。 |
| name | `xsd:string` | ズームターゲット名。 |
| xPosition | `xsd:int` | ズームターゲットのX位置。 |
| yPosition | `xsd:int` | ズームターゲットのY位置。 |
| 幅 | `xsd:int` | ターゲット幅をズームします。 |
| 高さ | `xsd:int` | ターゲットの高さをズームします。 |
| userData | `xsd:string` | Zoom ターゲットに関連付けられたユーザーデータ。 |
| 掲載順位 | `xsd:int` | アセットに関連付けられているズームターゲットを位置に基づいて並べ替えることができます |
| lastModified | `xsd:dateTime` | 最終変更日。 |

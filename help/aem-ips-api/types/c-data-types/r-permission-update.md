## パラメーター {#section-2fc9bea56b6d4b72b80d4f04c5f9b862}

| 名前 | 種類 | 説明 |
|---|---|---|
| groupHandle | `xsd:string` | グループハンドル。 |
| permissionType | `xsd:string` | 権限タイプ。 |
| isAllowed | `xsd:boolean` | 権限の更新を許可するかどうかを決定します。 |
| isOverride | `xsd:boolean` | 権限が別の権限を上書きするかどうかを決定します。 |

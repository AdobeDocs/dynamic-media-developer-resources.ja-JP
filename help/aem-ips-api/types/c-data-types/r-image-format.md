# ImageFormat{#imageformat}

画像出力設定の非構造化名前付きセット。

アセットタイプではありません。

## パラメータ {#section-2fc9bea56b6d4b72b80d4f04c5f9b862}

| 名前 | 種類 | 説明 |
|---|---|---|
| `*`imageFormatHandle`*` | `xsd:string` | 画像形式ハンドル。 |
| `*`name`*` | `xsd:string` | 画像形式名。 |
| `*`urlModifier`*` | `xsd:string` | 画像形式を表す画像サーバーのURL。 |

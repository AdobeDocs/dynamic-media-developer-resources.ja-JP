# CompanyMember{#companymember}

ユーザーが属する様々な会社を示します。

構文

## パラメータ {#section-03238fa4af3547f89faa1db91ccfd191}

| 名前 | 種類 | 説明 |
|---|---|---|
| `*`ユーザ`*` | `types:User` | ユーザー名。 |
| `*`role`*` | `xsd:string` | ユーザーが属する会社ごとに割り当てる役割。 |
| `*`isActive`*` | `xsd:boolean` | 所属する会社ごとに、ユーザーのステータスを設定します。 |

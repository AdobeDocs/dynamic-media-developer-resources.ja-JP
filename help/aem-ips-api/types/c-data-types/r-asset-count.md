# AssetCount{#assetcount}

会社のアセットタイプと数。

アセットタイプには、[!DNL ViewerSWf]、[!DNL PDF]、[!DNL Cabinet]などがあります。

| 名前 | 種類 | 説明 |
|---|---|---|
| ` *`assetType`*` | `xsd:string` | 数に関連付けられているアセットのタイプ。 |
| ` *`カウント`*` | `xsd:int` | 会社内の`assetType`アセットの数。 |

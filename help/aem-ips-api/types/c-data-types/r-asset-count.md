# [!DNL AssetCount]{#assetcount}

会社のアセットタイプとカウント。

アセットタイプには、[!DNL ViewerSWf]、[!DNL PDF]、[!DNL Cabinet] などがあります。

| 名前 | 種類 | 説明 |
|---|---|---|
| assetType | `xsd:string` | カウントに関連付けられているアセットのタイプ。 |
| カウント | `sd:int` | 会社内の `assetType` アセットの数。 |

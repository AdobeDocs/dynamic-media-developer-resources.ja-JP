# [!DNL AssetCount]{#assetcount}

会社のアセットタイプと数。

アセットタイプには、[!DNL ViewerSWf]、[!DNL PDF]、[!DNL Cabinet]などが含まれます。

| 名前 | 種類 | 説明 |
|---|---|---|
| assetType | `xsd:string` | カウントに関連付けられたアセットのタイプ。 |
| カウント | `sd:int` | 会社内の`assetType` アセットの数。 |

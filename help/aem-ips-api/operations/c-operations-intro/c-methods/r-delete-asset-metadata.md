# deleteAssetMetadata{#deleteassetmetadata}

アセットのメタデータ値を削除します。 メタデータ削除の配列を使用して値をバッチで設定します。

構文

## 認証済みユーザータイプ{#section-e913be43b684491daf02bc73211e4290}

* `IpsUser`
* `IpsAdmin`
* `IpsCompanyAdmin`
* `ImagePortalAdmin`
* `ImagePortalContrib`
* `ImagePortalContribUser`

>[!NOTE]
>
>ユーザーは、アセットに対する読み取りおよび削除のアクセス権を持っている必要があります。

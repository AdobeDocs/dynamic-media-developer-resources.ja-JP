# getAssets{#getassets}

画像実稼働システム（IPS）からアセットを返します。

構文

## 許可されているユーザータイプ {#section-4673c1c9f4314160af8b165eb2dd20cc}

* `IpsUser`
* `IpsAdmin`
* `IpsCompanyAdmin`
* `TrialSiteAdmin`
* `TrialSiteUser`
* `ImagePortalAdmin`
* `ImagePortalUser`
* `ImagePortalContrib`
* `ImagePortalContribUser`

>[!NOTE]
>
>ユーザーがアクセス権を持つアセットのみを返します。
